## 1. 현재 시스템 AI 역량 계층 (고도화 후 기준)

```
┌─────────────────────────────────────────────────────────────────┐
│ Layer 4 (신규): 매칭·지식축적·모멘텀 추적                         │
│   3-Stage 매칭엔진 · CasePack 8-Block · Bridge SM              │
│   프로모션 랭킹 · 크로스시스템 AI 브리핑 · 공실연동 신호          │
├─────────────────────────────────────────────────────────────────┤
│ Layer 3: 24+ AI 에이전트 (기존)                                  │
│   MemoParser · BuildingMiniTruth · BlindTeaser · DealCuriosity  │
│   SectionPlanner · 18-Section Writer · DraftGuardrails          │
│   VisualClassification · TenantFit · VibeFit · LeasingPage     │
├─────────────────────────────────────────────────────────────────┤
│ Layer 2: 데이터 인프라                                           │
│   Supabase SSoT · Gate 시스템(G0-G5) · RLS · activity_events   │
│   match_results · deal_casepacks · deal_pipeline_states         │
├─────────────────────────────────────────────────────────────────┤
│ Layer 1: 규제 안전장치                                           │
│   P0 금지패턴 · DisclosureGuard · 면책문구 자동삽입              │
│   BoundaryNote · 8개 민감필드 자동마스킹                          │
└─────────────────────────────────────────────────────────────────┘
```

**현재 등급: 4단계 (지식 축적 AI)**
- 딜 결과가 CasePack으로 구조화 → 유사 딜 브리핑 자동 제공
- 매칭 결과가 프로모션 점수에 피드백
- 아직 **자율 학습은 없음** (LLM 호출 기반, 모델 가중치 미변경)

## 8. 결론: 독점 공식의 진화

```
[고도화 전]  범용 LLM × 빠른 UX = "똑똑한 도구" (복제 6개월)

[고도화 후]  매칭엔진 × CasePack × 파이프라인 = "축적하는 도구" (복제 2년)

[R&D 융합]  특화 모델 × 특허 × 데이터 플라이휠 × 자기강화
            = "카테고리를 정의하는 플랫폼" (복제 5년+)
```

> **핵심 인사이트**: 고도화만으로 기업가치 9-12배 상승이 가능하지만,
> R&D 융합 시 추가 5-7배가 가능합니다.
> 그리고 가장 중요한 차이는 **해자 지속기간**입니다:
> - 도구 = 6개월 해자
> - 축적 = 2년 해자
> - **생태계 = 5년+ 해자 (복리 효과)**
