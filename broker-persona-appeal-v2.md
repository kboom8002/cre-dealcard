# 60대 꼬마빌딩 중개인 — 3시스템 소구 분석 + 활용 시나리오

---

## 대상 프로파일

| 항목 | 특성 |
|---|---|
| 연령대 | 55-70세 |
| 주력 매물 | 꼬마빌딩 50억-200억, 5-10층 이하 |
| 주요 고객 | 자산가 건물주, 법인 매수자 |
| 페인포인트 | 메모→워드→카카오 과정 느림, 복잡한 입력 거부 |
| 핵심 니즈 | 빠른 딜 클로징, 건물주 신뢰, 매수자 확보 |
| 기술 수준 | 카카오톡·사진 정도 익숙 |

|---|---|---|
| **DealCard** | 메모 1줄 → AI가 8개 항목 자동 정리 | 엑셀 2시간 → **2분** |

### P3. "맞는 매수자를 찾기 어려워"

| 시스템 | 소구 포인트 | 경쟁 방식 대비 |

| 가격 검증 | "실거래가 통계로 가격 근거 확보" | 건물주 협상력↑ |

## 3. 활용 시나리오

### 시나리오 A: "오늘 본 매물, 오늘 보낸다"

> 김 실장 (62세) — 오전에 성수동 근생 80억 매물을 봤다

```
09:30  현장 방문, 사진 10장 촬영
       → 카카오에 "성수동 5층 근생 80억대 올 공실없음 1층 카페"라고 메모

10:00  DealCard에 메모 붙여넣기
       → AI가 2분 만에 딜카드 생성:
         "성수동 / 근린상가 / 80억대 / 5층 / 공실 없음 / 1층 카페 입점"
       → 숨은 정보: "1층 카페 매출 안정 → 임대차 리스크 낮음"
       → 딜 호기심 점수: 78/100

10:05  모바일 IM 자동 생성 (7섹션)
       → "공유하기" → 카카오톡으로 매수자 후보 3명에게 발송

10:30  매수자 이 부장에게서 답장: "자세히 보고 싶습니다"
       → DealCard에서 "매수자 등록" → 조건 메모 입력
       → AI 매칭 실행: "S등급 — 법인 사옥 목적, 예산 적합, 성수동 선호"

11:00  딜 인텔리전스 확인:
       • 성사 확률: 72%
       • 같은 매수자가 관심 가진 다른 매물: 성수 사옥 65억
       • 유사 딜 패턴: "성수 근생에서는 사옥 전환 가능성 강조가 효과적"

      → 미팅 일정 잡기
```

**체감**: 과거 반나절 걸리던 "매물 정리 → 자료 작성 → 발송"이 **30분에 끝남**

### 시나리오 B: "매수자가 진지하니까 IM 올려보자"

> 박 사장 (67세) — 매수자 미팅 후 Full IM이 필요한 상황

```
[미팅 전]
  DealCard에서 "Full IM 만들기" 클릭
    → 모바일 IM 데이터 자동 이관
    → FullIM Readiness 점수: 62/100
    → 부족 항목: "임대차 계약서, 등기부등본, 관리비 내역"

[서류 보완]
  건물주에게 서류 요청 → 업로드
    → Readiness 82/100으로 상승
    → AI가 18섹션 초안 자동 생성 (10분)

[AI가 잡아주는 차별화 섹션]
  섹션3 투자 논거: "법인 사옥 목적 매수자에게 세제 혜택 강조"
    (← P-D2 클러스터 기반, 이 매수자가 '법인사옥 추구형'이므로)

  섹션7 미시 시장: "AiPage에서 분석한 결과, 1층 카페 공간에
    3건의 임대 문의가 있으며, AI 적합도 점수 85점"
    (← AiPage 임차인 분석 데이터 자동 삽입)

  섹션12 가치평가: "국토부 실거래 기준 성수동 근생
    최근 1년 23건 → AI 추정가 72-88억 (참고용)"
    (← P-D 가격 예측 자동 삽입)

[안전장치]
  ⚠️ Risk Gate: "수익률 보장" 문구 발견 → 자동 수정 제안
  ⚠️ Disclosure Gate: 임차인 이름 노출 → 업종으로 자동 마스킹

[전문가 패치]
  "섹션10 NOI 분석은 CRE 컨설턴트 검토 필요" → 전문가에게 연결

[최종]
  Buyer-Ready 승인 → PDF 다운로드 → 매수자에게 전달
```

**체감**: "IM 만들라고 하면 막막했는데, AI가 초안 잡아주고 뭐가 부족한지 알려주니 할 만해"

> 최 부장 (58세) — 위탁받은 건물 3채의 현황을 건물주에게 보고

```
[AiPage 대시보드]
  건물 A (성수동 5층):
    • 공간 3개 관리 중
    • 2층 공실 → AI 추천 업종: "소규모 사무실 또는 스튜디오"
    • 이번 주 임대 문의 5건 (AI 진지도: 진지 2건, 탐색 3건)
    • 사진 25장 → AI가 외관 5, 내부 15, 설비 5장으로 자동 분류

  건물 B (강남 4층):
    • 전 층 임대 중, 공실 없음
    • 임차인 계약 만기 2개월 전 알림

  건물 C (합정 3층):
    • DealCard 매각 파이프라인: "매수자 미팅" 단계
    • 성사 확률 68%
    • 매칭된 매수자: 3명 (S등급 1명)

[건물주 보고서 자동 생성]
  "건물주님, 이번 주 보고드립니다.
   • 성수동: 공실 공간에 진지한 문의 2건 들어왔습니다.
     AI 분석 결과 소규모 사무실 업종이 가장 적합합니다.
   • 합정: 매각 진행 중이며, 최우선 매수자와 미팅 예정입니다.
     현재 딜 성사 가능성은 68%입니다."
  → 카카오로 건물주에게 전달
```

**체감**: "매주 전화로 이것저것 설명하느라 30분씩 걸렸는데, 리포트 보내면 5분이면 끝"

## 4. 한 줄 요약 — 왜 이 시스템인가

| 기존 방식 | 이 시스템 |
|---|---|
| 메모 → 엑셀 → 워드 → 카카오 (반나절) | **메모 → AI → 카카오 (2분)** |
| 매수자 찾기: 내 머릿속 + 전화 | **AI 매칭 + 자동 추천** |
| IM: 워드로 8시간 작업 | **AI 초안 10분 + 검토 1시간** |
| 가격: "감으로" 협상 | **실거래 데이터 기반 근거** |
| 건물주 보고: 전화 30분 | **AI 리포트 카카오 발송 5분** |
| 딜 경험: 내 머릿속에만 | **시스템이 기억하고 학습** |

> **"복잡한 건 AI가 하고, 사장님은 사람 만나는 데 집중하세요"**
