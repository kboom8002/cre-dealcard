## ✅ 신규 빌드/배포 없이 데모 가능한가?

### 결론: **Yes, 가능합니다.**

현재 credeal.net은 커밋 `c794840 fix: inject Supabase Authorization token headers`로 배포되어 있습니다.

## 시나리오 #1 — 카톡 메모 → 1분 딜카드 생성

**소요 시간**: 4분  
**계정**: `demo-broker@credeal.net`  
**핵심 메시지**: *"카카오에 치던 메모가 30초 만에 법인 자산이 됩니다"*

### 단계

1. **접속**: https://credeal.net/broker/deal-card/new

2. **메모 붙여넣기** (클립보드에 사전 복사):
   ```
   성수동 꼬마빌딩, 대지 150평, 연면적 280평, 지상 4층
   1층 카페 임차 중 (보증 5천/월세 400), 2~4층 사무실 공실
   엘리베이터 없음, 2018년 전면 리모델링, 주차 3대
   매도인 양도세 이슈로 급히 팔고 싶어 함
   호가 78억, 협의 가능. 주소는 블라인드 처리 원함
   ```

3. **"딜카드 생성" 클릭** → AI 처리 대기 (~15~25초)

4. **결과 화면에서 확인할 포인트**:
   - 🔒 숨긴 정보 카드: `정확한 주소`, `임차인명`, `임대 단가`, `매도인 사정` 표시
   - ✅ 딜 포인트: 5개 이상 자동 생성
   - ⚠️ 유의사항: "양도세 이슈는 전문가 확인 권고"로 자동 치환
   - 카카오 공유 문구 복사 버튼

5. **나레이션 포인트**:
   > "양도세 이야기가 사라진 이유가 있습니다. 중개인이 세금 이슈를 확정적으로 언급하면 법적 책임 문제가 생길 수 있거든요. 시스템이 자동으로 '전문가 확인 권고'로 바꿔줬습니다."

**탭 A** (`demo-public@credeal.net`):
1. https://credeal.net/broker/deal-card/e2e00000-0001-0001-0001-000000000001 접속

   - 성수 꼬마빌딩: `gate_requested` 단계 (G1 대기)
