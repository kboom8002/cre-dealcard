## 시나리오 #14 — 브로커 콘텐츠 스튜디오 (뉴스레터 & 공동중개)

**소요 시간**: 4분  
**계정**: `demo-broker@credeal.net`  
**URL**: https://credeal.net/broker/studio  
**핵심 메시지**: *"딜카드를 넘어 — 중개인의 콘텐츠 & 협업 허브"*

### 4가지 기능 시연

**① 뉴스레터 큐레이션 편집기 (F1)**
1. 딜카드 2건 중 1건 클릭 (파란 테두리 활성)
2. 뉴스 항목 1건 추가
3. 큐레이션 조합 완성

**② AI 말투 코멘터 (F6)**
```
입력: "성수동 리모델링 매물 강추합니다"
→ AI 생성 클릭
→ 출력 예시:
  "김대표님께 — 최근 성수 권역 지산 인근 근생빌딩 거래량 전월 대비 +35%...
   관심 두신 80억대 매물은 사옥 실수요+밸류애드 스토리 최상의 포지션..."
```

**③ 화이트라벨 공유 링크 빌더 (F4)**
1. 서브도메인 입력: `demo-broker`
2. 테마 색상 선택
3. "화이트라벨 모바일 페이지 생성" → URL 생성
4. "카톡 발송" 버튼

**④ 공동중개 제안 시스템 (D1)**
1. 수수료 분배: `50:50 공동중개` 선택
2. 수신 브로커: `전체 브로커 공개 제안` 선택
3. 메시지 입력: `성수 사옥 70~80억 법인 매수자 확보. 리모델링 근생 보유 중개인 매칭 요청.`
4. "공동중개 제안 등록" 버튼

### 나레이션 포인트

> "AI 말투 코멘터를 보시면, 제가 짧게 '강추합니다'라고 쓴 메모를 AI가 시장 데이터와 고객 맥락을 결합해 전문가 화법으로 바꿔줬습니다. 고객에게 보낼 카카오 문구를 이렇게 완성합니다."
