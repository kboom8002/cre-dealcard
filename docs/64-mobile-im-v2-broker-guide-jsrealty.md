## 1. 모바일 IM은 왜 필요한가?

### 중개인이 겪는 현실적 문제

| 상황 | 기존 방식 | 문제점 |
|------|---------|--------|
| 매물 문의가 왔을 때 | PPT 파일을 이메일로 전송 | 카카오톡에서 바로 못 봄, 열리지 않는 경우 많음 |
| 투자자에게 빠르게 전달할 때 | 네이버 블로그 링크 공유 | 전문성 떨어짐, 수익률 분석 없음 |
| Full IM을 만들 때 | 외주 의뢰 (50~200만원, 2~3주) | 비용 과다, 시간 오래 걸림 |
| 매수자가 여러 매물 비교할 때 | 각각 다른 형식의 자료 | 비교 불가, 의사결정 지연 |

### 모바일 IM이 해결하는 것

> **"카카오톡으로 보내면 5초 안에 열리는, AI가 자동으로 만들어주는 전문 투자설명서"**

- 📱 **모바일 최적화**: 카카오톡/링크로 공유하면 즉시 열람 가능
- ⏱️ **30초 생성**: 주소와 월세만 입력하면 AI가 7섹션 자동 작성
- 💰 **무료**: 외주 비용 없이 무제한 생성
- 📊 **자동 재무 분석**: Cap Rate, NOI, IRR, DCF 자동 계산
- 🏢 **공공데이터 자동 연동**: 건축물대장, 공시지가, 용도지역 자동 조회

## 2. 기존 IM 방식과 비교

### PPT 단순 버전 IM vs 크리딜 모바일 IM vs Full IM

| 항목 | PPT 단순 IM | 크리딜 모바일 IM (v2) | Full IM (30~50p) |
|------|-----------|---------------------|-----------------|
| **제작 시간** | 2~5시간 | ⚡ **30초** | 2~3주 |
| **제작 비용** | 무료 (직접) | ⚡ **무료** | 50~200만원 |
| **재무 분석** | ❌ 수동 계산 | ✅ **자동** (Cap Rate, NOI, IRR, DCF) | ✅ 전문가 수동 |
| **공공데이터** | ❌ 수동 조회 | ✅ **자동 연동** (건축물대장, 공시지가) | ✅ 수동 첨부 |
| **모바일 열람** | ❌ PPT 파일 | ✅ **즉시 열람** (카카오톡 링크) | ❌ PDF 다운로드 |
| **카카오톡 공유** | ❌ 파일 첨부 | ✅ **전문 카드** (OG 이미지+재무지표) | ❌ 불가 |
| **투자자 반응 추적** | ❌ 불가 | ✅ **실시간** (조회수, 체류시간, Warm Call) | ❌ 불가 |
| **수정/업데이트** | 매번 재작성 | ✅ **즉시 재생성** | 재의뢰 필요 |
| **정보 신뢰도 표시** | ❌ 없음 | ✅ **출처 배지** (공부확인/중개인입력/AI추정) | ✅ 출처 주석 |
| **매물 마스킹** | 수동 편집 | ✅ **자동** (blindName 마스킹) | 수동 편집 |
| **레버리지 분석** | ❌ 없음 | ✅ **자동** (자기자본/보증금/대출 비율) | ✅ 수동 |
| **DCF 감응도** | ❌ 없음 | ✅ **3×3 히트맵** | ✅ 수동 엑셀 |

### 핵심 차별점 요약

> 🎯 **PPT보다 100배 빠르고, Full IM의 80%를 무료로 자동 생성**

## 3. 모바일 IM v2 신규 기능

### 3-1. 📊 DCF 감응도 히트맵 *(신규)*

투자자가 가장 궁금해하는 질문: **"할인율과 매각 배수가 달라지면 가치가 얼마나 변하나?"**

- 3×3 매트릭스로 할인율(Cap Rate) × 매각 배수별 NPV 자동 계산
- 색상 히트맵으로 한눈에 파악 (녹색=양호, 빨강=위험)
- **입력값**: 매각 희망가, 월 임대료 (바텀시트에서 입력)
- **기존**: 수동 엑셀 작업 → **자동 생성**

### 3-2. 💰 레버리지 자금 구조 차트 *(신규)*

대출을 활용한 투자 시 실제 필요 자기자본과 레버리지 수익률 시각화.

- 자기자본 / 보증금 / 대출 비율 차트
- 레버리지 수익률 자동 계산
- **입력값**: 대출 잔액, 보증금 (바텀시트에서 입력)

### 3-3. 🏷️ 출처 배지 시스템 *(신규)*

각 정보의 출처를 투명하게 표시하여 매수자 신뢰도 향상.

| 배지 | 의미 | 예시 |
|------|------|------|
| ✓ 공부 확인 | 건축물대장, 공시지가 등 공공데이터 기반 | 연면적, 사용승인일, 공시지가 |
| 👤 중개인 입력 | 중개인이 직접 입력한 데이터 | 월 임대료, 공실률, 매각가 |
| ⚙ AI 추정 | AI가 계산하거나 추론한 데이터 | Cap Rate, IRR, 투자 메리트 |

### 3-4. 🏆 정보 완성도 A-D 등급 *(개선)*

입력한 데이터에 따라 IM 품질 등급이 결정됩니다.

| 등급 | 조건 | 효과 |
|------|------|------|
| **A등급** 🟢 | 주소 + 공공데이터 + 월세 + **매각가** | Cap Rate, IRR, DCF 모두 산출 → **투자 검토 가능** |
| **B등급** 🟡 | 주소 + 공공데이터 + 월세 | 기본 수익률만 산출 |
| **C등급** 🟠 | 주소 + 공공데이터 | 건물 정보만 표시 |
| **D등급** 🔴 | 그 외 | 데이터 보충 필요 |

> 💡 **팁**: 매각 희망가를 입력하면 B→A 등급으로 승급되어, 투자자가 바로 검토할 수 있는 수준의 IM이 됩니다.

### 3-5. 💬 카카오톡 전문 공유 *(개선)*

투자자에게 카카오톡으로 공유할 때 전문적인 카드가 표시됩니다.

**개선 전:**
```
제목: 투자 매물
설명: AI 자동 생성 모바일 투자설명서
버튼: 투자설명서 보기
```

**개선 후:**
```
제목: [강남 GBD] *** 오피스빌딩
설명: 매각 희망가 50억대 · Cap Rate 4.2% · 크리딜 프리미엄 투자설명서
버튼: 투자설명서 즉시 열람
이미지: 1200×630 OG 이미지 (권역, 가격대, Cap Rate 표시)
```

### 3-6. 🔒 데이터 투명성 강화 *(개선)*

- 공공데이터 API 실패 시 이전에는 **가짜 데이터**("강남 센트럴타워" 등)가 IM에 포함될 수 있었음
- v2부터 **API 실패 시 해당 데이터 미포함** → 투자자 오도 방지
- 승인일자(`approvedAt`)가 뷰어에 정상 표시

## 4. Full IM으로의 확장성

크리딜 모바일 IM은 Full IM의 **전단계(Pre-IM)** 역할을 합니다.

### 자동 전환 가능 항목

| 모바일 IM 데이터 | Full IM 확장 시 | 확장 방법 |
|----------------|---------------|---------|
| 7섹션 마크다운 | 30~50p 보고서 | 섹션별 심화 분석 추가 |
| Cap Rate / NOI | DCF 상세 모델 | 10년 현금흐름표 생성 |
| 건축물대장 요약 | 건축물대장 원본 첨부 | PDF 자동 연동 |
| 사진 3~12장 | 사진 + 도면 + 평면도 | 파일 업로드 확장 |
| 출처 배지 | 감정평가 보고서 연동 | 전문가 검증 확장 |
| 임대 현황 요약 | 렌트롤 상세표 | 엑셀 렌트롤 임포터 연동 |

### 현재 구현된 확장 인프라

1. **바이브 카드 연동**: IM → 중개인 프로필 카드 → 상담 예약
2. **매거진 연동**: 주간 AI 매거진에 추천 매물로 자동 노출
3. **Warm Call 시그널**: 투자자가 IM을 60초 이상 볼 때 중개인에게 알림톡
4. **Full IM 수동 핸드오프**: "상세 투자설명서(IM) 요청" CTA → 중개인 직접 대응

### 향후 확장 로드맵

| 단계 | 기능 | 설명 |
|------|------|------|
| **Phase 1** (현재) | 모바일 IM Lite | AI 자동 7섹션 + DCF + 레버리지 |
| **Phase 2** (예정) | Full IM 자동화 | 렌트롤 상세표 + 도면 + 감정평가 연동 |
| **Phase 3** (예정) | 영문 IM | 외국인 투자자 대응 (자동 번역) |
| **Phase 4** (예정) | IM → 계약서 | 투자설명서 → LOI(투자의향서) 자동 생성 |

## 5. 활용 시나리오

### 시나리오 A: 카카오톡으로 매수 문의 응대

```
매수자: "강남에 50억대 꼬마빌딩 있으면 보내주세요"

중개인: (크리딜에서 30초 만에 IM 생성)
        → 카카오톡으로 전문 카드 공유
        → 매수자가 5초 만에 열람
        → Cap Rate 4.2%, NOI 2.1억 확인
        → "상세 IM 요청" 클릭
        → 중개인에게 Warm Call 알림

소요 시간: 30초 (기존: 2~5시간)
```

### 시나리오 B: 주간 매거진과 연동

```
1. 크리딜에서 금주 신규 매물 3건의 IM 자동 생성
2. 주간 AI 매거진에 추천 매물로 자동 노출
3. 구독자 50명에게 자동 발송
4. 구독자가 매거진에서 "IM 열람 신청" 클릭
5. 중개인에게 리드 수신

전환률: 매거진 → IM 열람 → 상담 요청
```

### 시나리오 C: 경쟁 매물 비교

```
매수자가 3개 매물 비교 검토 중:
- 매물 A: Cap Rate 4.2%, DCF NPV 45억
- 매물 B: Cap Rate 3.8%, DCF NPV 42억
- 매물 C: Cap Rate 5.1%, DCF NPV 51억

모든 IM이 동일 포맷 → 즉시 비교 가능
기존: 각각 다른 PPT 형식 → 비교 불가
```

## 6. 고도화 개선 포인트 (피드백 요청)

다음 항목에 대한 제이에스부동산 중개인분들의 의견을 반영하여 개선하겠습니다.

### 🔍 검토 요청 항목

| # | 개선 포인트 | 현재 상태 | 질문 |
|---|-----------|---------|------|
| 1 | **렌트롤 상세표** | 현재: 월세 총액만 입력 | 층별 렌트롤(임차인, 면적, 단가)을 IM에 포함할 필요가 있나요? |
| 2 | **도면/평면도** | 현재: 사진만 지원 | 평면도 PDF 첨부 기능이 필요한가요? |
| 3 | **감정평가 연동** | 현재: AI 추정 기반 | 감정평가 보고서 데이터를 연동하면 신뢰도가 올라갈까요? |
| 4 | **투자자 조회 분석** | 현재: Warm Call만 | 어떤 섹션을 오래 봤는지, 몇 번 재방문했는지 분석 리포트가 필요한가요? |
| 5 | **IM 비교 대시보드** | 현재: 개별 IM만 | 같은 권역 매물 3~5건을 한 화면에서 비교하는 기능이 유용한가요? |
| 6 | **영문 IM** | 현재: 한국어만 | 외국인 투자자 대응을 위한 영문 IM 자동 번역이 필요한가요? |
| 7 | **프라이빗 IM 잠금** | 현재: 공개 URL | NDA 서명 후에만 열람 가능한 프라이빗 IM이 필요한가요? |
| 8 | **섹션 커스터마이징** | 현재: 7섹션 고정 | 중개인이 섹션을 추가/삭제/순서 변경할 수 있으면 좋겠나요? |
| 9 | **AI 톤 앤 매너** | 현재: 전문 분석 스타일 | 더 부드럽거나 공격적인 톤으로 변경할 수 있으면 좋겠나요? |
| 10 | **오프라인 PDF 출력** | 현재: 모바일 웹만 | IM을 PDF로 출력하여 오프라인 미팅에서 사용할 필요가 있나요? |

## 7. 바로 시작하기

### Step 1: 로그인
- https://www.credeal.net/login 접속
- 기존 계정으로 로그인 (중개인 권한 자동 부여)

### Step 2: 딜카드 선택
- 대시보드에서 기존 딜카드 선택 또는 신규 딜카드 생성

### Step 3: 모바일 IM 생성
- "📝 프리미엄 투자설명서 만들기" 클릭
- 주소, 월세, 매각가 입력 (매각가 입력 시 A등급!)
- 사진 3장 이상 업로드 (선택)
- "📄 프리미엄 투자설명서 만들기" 버튼 클릭
- **30초 후 완성!**

### Step 4: 승인 및 공유
- IM 내용 확인 후 "승인 및 공개"
- 카카오톡으로 투자자에게 즉시 공유
