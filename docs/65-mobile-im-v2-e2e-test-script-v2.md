```
서울 중구 필동3가 44-5 근생빌딩 월세 3800만 관리비 200만 보증금 8억

```

## 테스트 실행 방법

### 방법 1: 프로덕션 UI 테스트
1. credeal.net 로그인
2. 딜카드 생성 (메모 입력)
3. 딜카드 상세 → "모바일 투자설명서 작성" 클릭
4. 바텀시트에서 위 데이터 입력
5. "IM 생성" 클릭
6. 검토 화면에서 검증 항목 확인
7. 승인 → 공개 IM 페이지 확인
