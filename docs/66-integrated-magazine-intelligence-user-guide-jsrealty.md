## 2. 로그인 & 대시보드 첫 화면

### 2-1. 로그인

1. 브라우저에서 **https://www.credeal.net/login** 접속

### 2-2. 대시보드 구성

로그인 후 대시보드 화면에는 다음 탭들이 표시됩니다:

|----|------|------|

| **🌅 모닝 정보** | 🌅 | AI 마켓 브리핑 + 실거래 + 경매 + 임대시장 |

| **📊 리포트** | 📊 | 성과 분석 리포트 |

> 💡 **팁**: "모닝 정보" 탭이 아침 업무의 시작점입니다. 매일 아침 이 탭부터 확인하세요!

## 3. 오늘의 정보 (모닝 인텔리전스)

### 3-1. 개요

매일 아침 8시에 자동 업데이트되는 **AI 기반 시장 브리핑** 서비스입니다.  
AI가 공공데이터, 뉴스, 실거래 데이터를 분석하여 **오늘 영업에 바로 쓸 수 있는 인사이트**를 제공합니다.

### 3-2. 3가지 모드

오늘의 정보에는 상단에 3개 탭이 있습니다:

#### 📡 HQ 브리핑 (기본 탭)

크리딜 AI가 자동으로 생성하는 **본사 발행 브리핑**입니다.

| 카드 | 내용 |
|------|------|
| **AI 마켓 에디터 브리핑** | 시장 뉴스·실거래·경매·임대·내 매물 데이터를 종합한 5~7줄 브리핑 |
| **내 매거진 성과 카드** ⭐NEW | 최근 발행한 매거진의 평균 조회수 + 활성 구독자 수 표시 |
| **내 매물 vs 실거래** | 내 보유 매물과 최근 실거래가 비교 |
| **오늘의 실거래 체결** | 권역 내 최신 거래 내역 |
| **경매 & 공매 신건** | 대법원 경매 정보 자동 수집 |
| **임대 & 공실 현황** | 권역별 공실률, 임대료 동향 차트 |
| **시장 심리 지수** | 뉴스 감성 + 거래량 + 경매 낙찰률 기반 복합 지수 (0~100) |

> 🎯 **v3 신규**: "내 매거진 성과 카드"가 추가되어, 발행한 매거진의 독자 반응을 대시보드에서 바로 확인할 수 있습니다. "상세 성과 보기"를 누르면 매거진 에디터의 성과 탭으로 이동합니다.

#### ✍️ 마이 인텔리전스

중개인이 **직접 수집한 정보를 AI가 정리**해 주는 탭입니다.

**사용법:**

1. 텍스트 영역에 뉴스 기사, 시장 메모, 거래 정보 등을 **복사+붙여넣기**
2. **"AI 정리"** 버튼 클릭
3. AI가 항목별로 자동 분류 → **요약 + 임플리케이션 + 종합 인사이트 + 액션 아이템** 제공

```
📋 예시 입력:
1. 강남 테헤란로 이면 근생 빌딩 85억 거래 체결
2. 성수 F&B 공실률 1.8%로 하락, 임대 수요 지속
3. 한국부동산원 발표: 서울 오피스 공실률 2.3%

↓ AI 정리 ↓

📊 종합 인사이트:
"강남 GBD와 성수 권역 모두 매수 수요가 견조합니다.
 특히 F&B 공실률 하락은 임대 수익 안정화를 의미합니다."

🎯 액션 아이템:
→ F&B 임차인 대상 성수 매물 안내 우선
→ 강남 85억 거래를 레퍼런스로 가격 조정 검토
```

#### 🎨 커스텀 결합

HQ 브리핑과 마이 인텔리전스를 **AI가 합성**하여 나만의 브리핑을 생성합니다.

**사용법:**
1. HQ 브리핑 확인 ✅
2. 마이 인텔리전스 입력 & AI 정리 ✅
3. "커스텀" 탭 → **"AI 커스텀 브리핑 생성"** 클릭
4. 결과를 편집(선택) → **"매거진 편집하기"** 클릭하면 자동으로 매거진 에디터로 이동

### 3-3. ⭐ 매거진 추가 기능 (v3 신규)

오늘의 정보에서 발견한 좋은 정보를 **원클릭으로 매거진 임시저장함에 추가**할 수 있습니다.

**추가 가능한 항목:**

| 카드 종류 | 추가 버튼 위치 | 저장 방식 |
|----------|-------------|----------|
| **오늘의 실거래 체결** | 각 거래 카드 우측 하단 `📰 매거진 추가` | 드래프트에 자동 저장 |
| **경매 & 공매 신건** | 각 경매 카드 우측 하단 `📰 매거진 추가` | 드래프트에 자동 저장 |
| **AI 커스텀 브리핑** | 히어로 영역 `매거진 편집` 버튼 | 드래프트 + 에디터 이동 |

> 💡 **드래프트 뱃지**: "매거진 편집" 버튼 우측 상단에 현재 임시저장된 건수가 **숫자 뱃지**로 표시됩니다. 예: `📰 매거진 편집 ③`

### 3-4. 권역 선택

화면 상단에서 관심 권역을 선택할 수 있습니다:

| 코드 | 권역 | 포함 지역 |
|------|------|----------|
| **성수** | 성동구 | 성수동, 뚝섬, 서울숲 |
| **GBD** | 강남구 | 강남, 역삼, 테헤란, 삼성동 |
| **YBD** | 영등포구 | 여의도, 영등포 |

### 3-5. 카카오톡 공유

HQ 브리핑 하단의 **공유 버튼**을 통해 매수자/투자자에게 시장 브리핑을 카카오톡으로 전송할 수 있습니다.

**공유되는 형태:**
```
[강남 GBD] 오늘의 시장 브리핑
📊 시장 심리 지수: 62/100 (보합)
Cap Rate 4.2% | 실거래 3건 체결
→ [브리핑 전문 보기] 버튼
```

## 4. 모바일 매거진 에디터 (All-in-One)

### 4-1. 개요

매거진 에디터는 중개인의 **전문 콘텐츠 제작 허브**입니다.  
이번 v3에서 기존 "콘텐츠 스튜디오"를 완전 흡수하여, **하나의 화면에서 기획·작성·AI보조·발행·성과분석**까지 모든 작업을 수행합니다.

### 4-2. 진입 방법

매거진 에디터에 진입하는 방법은 여러 가지입니다:

| 경로 | 방법 |
|------|------|
| **대시보드 → 모닝 정보** | 히어로 영역의 `📰 매거진 편집` 버튼 클릭 |
| **대시보드 → 더보기 메뉴** | 좌측 메뉴 → "매거진 에디터" 링크 |
| **정보 카드 → 매거진 추가** | `📰 매거진 추가` 버튼 → 자동 이동 |
| **직접 URL** | `/broker/magazine-editor` 접속 |
| **기존 스튜디오 URL** | `/broker/studio` 접속 → 자동 리다이렉트 |

### 4-3. 8개 탭 상세 안내

에디터 상단에 가로 스크롤 가능한 8개 탭이 있습니다. 좌우로 스와이프하여 모든 탭에 접근하세요.

---

#### 📰 탭 1: 커버 (Cover)

매거진의 **첫인상**을 결정하는 표지 설정 탭입니다.

**설정 항목:**

| 항목 | 설명 | 예시 |
|------|------|------|
| **헤드라인** | 이번 주 매거진의 제목 | "성수 F&B 공실률 역대 최저, 지금이 진입 시점" |
| **시장 온도계** | 5단계 중 현재 시장 판단 선택 | 적극 매수 🔥 / 선별 매수 📈 / 관망 ⏸️ / 조정 대기 📉 / 위기 경계 🚨 |
| **커버 키워드** | 3개 이하 핵심 키워드 | `#성수동` `#F&B` `#공실률하락` |
| **테마 컬러** | 매거진 전체 색상 톤 | 인디고(기본), 에메랄드, 로즈 등 |
| **커버 이미지** | 매거진 표지 이미지 업로드 | 현장 사진, 건물 사진 등 |

> 💡 **팁**: 헤드라인에 **구체적인 숫자**를 넣으면 독자 클릭률이 높아집니다.  
> 예) ❌ "성수 시장 분석" → ✅ "성수 공실률 1.8%, 3년 내 최저"

---

#### ✏️ 탭 2: 필드노트 (Field Note)

중개인의 **현장 경험**을 독자와 공유하는 탭입니다. AI가 대신 쓸 수 없는 **차별화 콘텐츠**입니다.

**입력 항목:**

| 필드 | 설명 | 작성 팁 |
|------|------|---------|
| **주간 시장 요약** | 이번 주 시장 한 줄 요약 | "강남 30~50억 구간 매수 문의 3건 → 시장 회복 신호" |
| **매수자 반응** | 실제 매수자 피드백 | "30대 스타트업 대표, 성수 F&B 매물에 관심" |
| **매도자 반응** | 매도자 분위기 | "호가 5% 상향 조정, 급매 의사 없음" |
| **시장 판단** | 중개인 본인의 판단 | "하반기까지 보합세, 연말 상승 전환 예상" |
| **중개인 한마디** | 독자에게 전하는 메시지 | "지금은 급매를 찾는 것보다 입지를 먼저 확정하세요" |

> 🎯 **필드노트가 중요한 이유**: 매수자/투자자는 AI가 만든 뉴스보다 **현장 중개인의 생생한 코멘트**를 더 신뢰합니다. 이것이 매거진 구독을 유지시키는 핵심입니다.

---

#### 🎯 탭 3: 테마 & 매물 (Theme Deals)

이번 주의 **투자 테마**를 설정하고, 관련 매물을 큐레이션합니다.

**구성:**
1. **주간 테마 제목**: "이번 주 주목: 성수동 F&B 꼬마빌딩"
2. **테마 본문 (마크다운)**: 시장 분석 + 근거 + 전망
3. **관련 자산 유형 태그**: `오피스`, `근린생활시설`, `다가구` 등
4. **추천 매물 선택**: 내 보유 딜카드에서 매물 토글 선택

> 💡 **팁**: 매물은 3건 이하로 엄선하세요. 너무 많으면 독자가 집중하지 못합니다.

---

#### 📰 탭 4: 뉴스 큐레이션 (News)

AI가 자동 수집한 시장 뉴스 중 매거진에 포함할 뉴스를 **선별**합니다.

**사용법:**
1. 뉴스 목록이 중요도 순으로 표시됩니다
2. 각 뉴스의 **토글 스위치**를 눌러 포함/제외를 결정합니다
3. 포함된 뉴스만 매거진에 표시됩니다

**뉴스 카드에 표시되는 정보:**
- ⭐ **중요도 점수** (1~10)
- 📂 **토픽 분류** (정책, 시장, 금리 등)
- 📰 **출처** (한국경제, 머니투데이 등)
- 📊 **감성** (긍정 / 부정 / 중립)

## 5. 모바일 IM (투자설명서)

### 5-1. 개요

모바일 IM은 **AI가 30초 만에 자동으로 생성하는 전문 투자설명서**입니다.  
기존 PPT 제작에 2~5시간 걸리던 작업을 원클릭으로 해결합니다.

### 5-2. IM 생성 절차

```
Step 1: 딜카드에서 매물 선택
Step 2: "📝 프리미엄 투자설명서 만들기" 클릭
Step 3: 주소, 월 임대료, 매각 희망가 입력
Step 4: 사진 업로드 (선택, 3장 이상 권장)
Step 5: "📄 투자설명서 만들기" 버튼 → 30초 후 완성!
```

### 5-3. IM에 포함되는 7개 섹션

| # | 섹션 | 내용 | 데이터 출처 |
|---|------|------|-----------|
| 1 | **📊 투자 요약** | Cap Rate, NOI, 핵심 지표 요약 | ⚙ AI 계산 |
| 2 | **🏢 건물 개요** | 연면적, 용도, 사용승인일, 주차 | ✓ 건축물대장 (공공데이터) |
| 3 | **💰 수익 분석** | 임대 수익, Cap Rate, NOI 상세 | 👤 중개인 입력 + ⚙ AI |
| 4 | **📈 DCF 감응도** | 할인율 × 매각 배수별 NPV 히트맵 | ⚙ AI 계산 |
| 5 | **🏗️ 레버리지 구조** | 자기자본 / 보증금 / 대출 비율 차트 | 👤 중개인 입력 |
| 6 | **📍 입지 분석** | 공시지가, 상권, 교통 | ✓ 공공데이터 |
| 7 | **🤖 AI 투자 메리트** | AI가 종합 분석한 투자 포인트 | ⚙ AI 분석 |

### 5-4. 정보 완성도 등급

입력 정보에 따라 IM 품질 등급이 결정됩니다:

| 등급 | 조건 | 효과 |
|------|------|------|
| **A 🟢** | 주소 + 공공데이터 + 월세 + **매각가** | Cap Rate, IRR, DCF 모두 산출 → 투자 검토 가능 |
| **B 🟡** | 주소 + 공공데이터 + 월세 | 기본 수익률만 산출 |
| **C 🟠** | 주소 + 공공데이터 | 건물 정보만 표시 |
| **D 🔴** | 그 외 | 데이터 보충 필요 |

> 🎯 **초심자 팁**: **매각 희망가를 반드시 입력**하세요! B→A 등급으로 올라가면 투자자가 바로 검토할 수 있는 IM이 됩니다.

### 5-5. 카카오톡 공유

IM 생성 후 카카오톡으로 공유하면 **전문 카드 형태**로 전송됩니다:

```
┌──────────────────────────────────┐
│  [강남 GBD] *** 오피스빌딩          │
│  매각 희망가 50억대 · Cap Rate 4.2% │
│  크리딜 프리미엄 투자설명서           │
│  ┌──────────────────────────────┐│
│  │  [투자설명서 즉시 열람]          ││
│  └──────────────────────────────┘│
└──────────────────────────────────┘
```

### 5-6. 매거진과의 연동

생성된 IM은 주간 매거진에서 **추천 매물**로 자동 노출됩니다.  
매거진 독자가 "투자설명서 보기 (7섹션)"을 클릭하면 해당 IM으로 이동합니다.

## 6. Vibe AI 명함

### 6-1. 개요

Vibe 명함은 중개인의 **AI 기반 디지털 프로필 페이지**입니다.  
단순한 연락처가 아니라, **전문성·투자 철학·보유 매물·최신 매거진**이 모두 담긴 프리미엄 비즈니스 카드입니다.

### 6-2. 접근 방법

- **관리 화면**: 대시보드 → 더보기 메뉴 → "Vibe 명함 관리"
- **공개 URL**: `https://www.credeal.net/vibe-card/나의슬러그`

### 6-3. Vibe 명함 구성 요소

| 섹션 | 내용 | 자동/수동 |
|------|------|----------|
| **🪪 히어로 카드** | 프로필 사진, 이름, 소속, VTI(투자성향), 3D 틸트 효과 | 자동 생성 |
| **👤 프로필 상세** | 전문 분야, 경력, 중개 실적, 면허번호 | 수동 편집 |
| **💼 보유 매물 IM** | 활성 딜카드 → IM Lite 바로가기 | 자동 연동 |
| **📰 최신 매거진** | 가장 최근 발행된 매거진 링크 + 시장 온도계 | 자동 연동 |
| **📮 구독 카드** | "이 중개인의 매거진 구독하기" CTA | 자동 삽입 |
| **📞 연락 & 공유** | 전화 걸기, 카카오톡/링크 공유 바텀시트 | 자동 |

### 6-4. VTI (Vibe Type Indicator)

AI가 중개인의 프로필과 거래 패턴을 분석하여 **투자 성향 유형**을 자동 매칭합니다.

예시 유형:
- **🏛️ 안정형 수호자**: 안정적 수익·장기 보유 선호
- **🚀 공격형 개척자**: 개발 잠재력·밸류애드 투자 선호
- **🎯 전략형 분석가**: 데이터 기반 의사결정·포트폴리오 최적화

> 💡 **VTI가 중요한 이유**: 매수자와 중개인의 VTI 상호보완성으로 최적 매칭이 이루어집니다. 프로필을 정확하게 작성할수록 AI 매칭 정확도가 올라갑니다.

### 6-5. Vibe 명함 관리

관리 화면에서 다음을 편집할 수 있습니다:

1. **프로필 사진 변경**: 카메라 아이콘 → 사진 업로드
2. **회사 로고 업로드**: 명함 하단에 표시되는 회사 로고
3. **파트너 로고 업로드**: 제휴사 로고 (선택)
4. **FAQ 관리**: 자주 묻는 질문 추가/삭제 (매수자가 자주 궁금해하는 내용)

### 6-6. 공유 방법

| 방법 | 설명 |
|------|------|
| **카카오톡 공유** | 전문 OG 카드 형태로 전송 |
| **링크 복사** | URL 복사 → 문자, 이메일 등에 붙여넣기 |
| **QR 코드** | 오프라인 미팅 시 스캔용 |

### 시나리오 B: 매수자로부터 카카오톡 문의 수신

```

중개인 대응 순서:
1. 크리딜에서 해당 매물 딜카드 → "프리미엄 투자설명서 만들기" (30초)
2. 완성된 IM → 카카오톡으로 전문 카드 공유 (5초)
3. 매수자: 카카오톡에서 터치 → 즉시 열람 (5초)

기존: PPT 2~5시간 제작 → 이메일 첨부 → 열리지 않는 경우 다수
```

### 시나리오 C: 주간 매거진으로 잠재 고객 발굴

```
1. 매주 금요일: 매거진 발행 (필드노트 + 추천 매물 3건)
2. 구독자 50명에게 자동 발송
3. 월요일 대시보드에서 확인:
   - 매거진 조회수 120회
```

## 9. 자주 묻는 질문 (FAQ)

### Q1. AI가 만든 매거진을 바로 발행해도 되나요?

> **아니요, 반드시 검토 후 발행하세요.** AI는 초안을 도와주지만, 전문성과 현장감은 중개인이 부여합니다. 특히 **필드노트**는 직접 작성하는 것이 독자 신뢰의 핵심입니다.

### Q2. 작성 중에 브라우저를 닫으면 내용이 사라지나요?

> **사라지지 않습니다.** v3부터 30초마다 자동 저장되므로, 다시 에디터에 접속하면 마지막 저장 시점의 내용이 복원됩니다.

### Q3. 기존 "콘텐츠 스튜디오" 메뉴는 어디 갔나요?

> 매거진 에디터에 **완전 통합**되었습니다. 기존 `/broker/studio` URL에 접속하면 자동으로 매거진 에디터로 이동합니다. 화이트라벨 설정은 "발행 설정" 탭에, 리드 스코어링은 "성과" 탭에 있습니다.

### Q4. Vibe 명함에 매물이 자동으로 뜨나요?

> **네.** 딜카드에서 "공개 시그널"로 설정한 매물이 Vibe 명함의 "보유 매물 IM Lite" 섹션에 자동 노출됩니다. 또한 최신 매거진도 자동으로 연동됩니다.

### Q5. 카카오톡으로 공유할 때 전문적인 카드가 안 뜨면?

> 카카오톡 공유는 **카카오 SDK**를 통해 전문 카드 형태로 전송됩니다. 첫 사용 시 카카오 앱이 최신 버전인지 확인하세요. 또는 "링크 복사" 방식으로 직접 URL을 공유할 수도 있습니다.

### Q6. 모바일 IM에서 매각 희망가를 공개하지 않으려면?

> IM 생성 시 **blindName(마스킹)** 기능이 자동 적용됩니다. 정확한 주소와 건물명 대신 "강남구 GBD *** 오피스빌딩"처럼 마스킹된 형태로 표시됩니다.

### Q7. 매거진 구독자는 어떻게 모으나요?

> 1. **Vibe 명함**에 자동 삽입된 "매거진 구독하기" 버튼
> 2. **모바일 IM** 하단의 구독 유도 카드
> 3. **매거진 공유 시** 열람 페이지 내 구독 버튼
> 4. 명함·이메일 서명에 Vibe 명함 URL 추가

### Q8. 한 주에 매거진을 여러 번 발행할 수 있나요?

> **가능합니다.** 다만, 주 1회 발행이 구독자 관리에 가장 효과적입니다. 긴급 시장 변동이 있을 때는 임시호(Extra Edition)를 발행하는 것도 좋은 전략입니다.
