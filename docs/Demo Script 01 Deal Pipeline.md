> **대상**: CRE 중개인 (현직 · 예비 회원)

> **"카톡 메모 한 줄이 전문 투자설명서와 매수자 매칭까지 이어지는 과정을, 실제 매물(미사랑 빌딩)로 체험합니다."**

## ACT 2.5. ⚡ 유니버설 메모 FAB — "아무 메모나 던지세요" (2분)

### 📍 화면: `https://www.credeal.net/broker` → 우측 하단 `+` 플로팅 버튼

> [!IMPORTANT]
> 이 기능은 DealCard의 **핵심 철학**을 보여줍니다 — "중개인은 메모만 하면 되고, 나머지는 AI가 한다."

#### 보여줄 것

1. **플로팅 `+` 버튼 클릭** → 하단 시트가 올라오면서 2가지 모드 선택:
   - 🎙️ **음성으로 말하기** — 마이크 버튼으로 즉석 음성 녹음, STT 자동 변환
   - ⌨️ **텍스트로 치기** — 직접 타이핑 또는 카톡 메모 붙여넣기

2. **AI 자동 분류** — 메모를 입력하면 AI가 내용을 분석하여 4가지 중 하나로 자동 라우팅:

   | AI 감지 유형 | 아이콘 | 자동 액션 |
   |-------------|--------|----------|
   | 🏢 **신규 매물 감지** | Building2 | → "딜카드 자동 생성하기" 버튼 |
   | 🎯 **매수자 조건 감지** | Search | → "매수 의향서 자동 등록" 버튼 |
   | ✏️ **기존 매물 정보 보강** | Edit3 | → 활동 기록에 자동 저장 |
   | 📝 **일반 메모** | CheckCircle | → 메모 아카이브에 저장 |

#### 🎤 시연 방법 (미사랑 빌딩 메모)

> **`+` 버튼**을 누르고 **텍스트 모드**를 선택합니다.
> 아래 내용을 붙여넣습니다:

```
숭실대입구역 코너 빌딩 매물입니다.
미사랑빌딩, 상도동 477-18, 2019년 신축 지하1 지상6
연면적 216평, 대지 68평, RC조 개별난방
매매가 112억 (135억에서 급조정), 융자 78억 신한
전층 만실이고 의료 빌딩이에요
B1 PT스튜디오 300만, 1층 약국 1100만, 2층 이비인후과 350만
3-4층 내과 714만, 5층 건강의학과 400만, 6층 상담연구소 100만
보증금 합계 57500만원, 월세 합계 2964만원, 연 3.4%
관리비 잉여수익 45만원 발생
40미터 30미터 코너 가시성 우수합니다
```

> AI가 **"신규 매물 감지"**로 분류하고 → **"딜카드 자동 생성하기"** 버튼이 나타남

#### 🎤 멘트 예시
> "현장에서 받은 건물 정보를 카톡에서 복사해서 **+ 버튼에 그대로 붙여넣기만** 하세요. AI가 자동으로 매물 정보를 감지하고 딜카드 생성으로 안내합니다."

#### ➡️ 다음 액션
