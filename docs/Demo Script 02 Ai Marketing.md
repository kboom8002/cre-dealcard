# 🎬 데모 스크립트 2 — "AI가 만든 마케팅 무기고: 브리핑에서 매거진까지"

> **대상**: CRE 중개인 (현직 · 예비 회원)  
> **소요시간**: 18~22분

## 🎯 이 데모의 목표

> **"매일 아침, AI가 고객에게 보낼 시장 브리핑과 나만의 부동산 매거진을 자동으로 준비하고, 건물 소유주에게는 매각 준비도 진단을 카카오톡으로 바로 전송하는 과정을 체험합니다."**

중개인의 **개인 브랜딩**, **고객 리텐션**, 그리고 **건물주 유입**을 동시에 해결하는 AI 마케팅 도구를 총망라합니다.

2. **권역별 탭 전환** — 성수 / 강남 GBD / 여의도 YBD 3개 권역

#### 🎤 멘트 예시

| 기존 방식 | DealCard 방식 |

## ACT 3. CRE 매거진 — 나만의 부동산 뉴스레터 (3분)

### 📍 화면: `https://www.credeal.net/magazine/{brokerId}/{date}`

> [!TIP]
> 대시보드의 **"매거진 공유"** 버튼으로 접근하거나, 직접 URL로 접속 가능합니다. 화이트라벨 슬러그가 설정되어 있다면 `magazine/{your-slug}/{date}` 형태의 **브랜딩된 고유 URL**로 접근됩니다.

#### 보여줄 것

1. **히어로 섹션** — "6월 16일 · 홍길동 중개인의 CRE 매거진" 타이틀, 프리미엄 다크 배경
2. **8섹션 매거진 구조**:
   - 🏢 **오늘의 주요 딜** — 내가 보유한 매물 하이라이트
   - 📰 **시장 뉴스 톱3** — AI 선별 기사 요약
   - 📊 **권역별 시세 동향** — 성수/강남/여의도 가격 변화
   - 💡 **투자 인사이트** — AI가 분석한 숨은 기회
   - 🏗️ **신규 공급 소식** — 분양/리모델링 정보
   - 📈 **금주의 거래 사례** — 실거래 하이라이트
   - 🎯 **딜카드 하이라이트** — 가로 스크롤 카드 슬라이더
   - 📞 **브로커 바이브카드 CTA** — 하단 고정, 전화/문의 버튼
3. **동적 OG 이미지** — 카카오톡 공유 시 1200×630px 맞춤 미리보기
4. **Web Share API** — "공유" 버튼으로 카톡/문자/이메일 선택

#### 🎤 멘트 예시
> "이 매거진은 **AI가 자동으로 생성**합니다. 여러분의 이름이 박힌 프리미엄 부동산 뉴스레터가 매일 아침 준비되고, 카카오톡으로 한 번 보내면 됩니다. 고객 입장에서 생각해보세요 — 매일 아침 전문적인 시장 분석을 보내주는 중개인이 있다면, 다른 곳으로 가시겠습니까?"

#### 🔗 연계 활용

```
모닝 인텔리전스 (AI 시장 분석)
        ↓ 데이터 기반
CRE 매거진 (프리미엄 콘텐츠)
        ↓ 카카오톡 공유 (화이트라벨 SDK 연동 완료)
고객의 신뢰 + 재방문
        ↓ 바이브카드 CTA
상담 문의 전환
```

- 매거진 하단에 **자동으로 내 바이브카드**가 삽입되어, 고객이 바로 연락할 수 있습니다.
- 딜카드 하이라이트를 터치하면 → 해당 딜카드의 IM Lite 페이지로 이동합니다.

## ACT 4. 🎬 콘텐츠 스튜디오 — 중개인의 콘텐츠 공장 (3분)

### 📍 화면: `https://www.credeal.net/broker/studio`

> [!TIP]
> 이번 고도화로 **브로커 홈 퀵액션에 "🎬 콘텐츠 스튜디오" 버튼**이 추가되어, 더 이상 "더보기" 메뉴를 뒤지지 않아도 바로 진입할 수 있습니다.

#### 보여줄 것

1. **상단 통계 바** — 내 총 딜 수, 활성 딜, 매칭 건수, 딜팩 현황
2. **8가지 콘텐츠 도구** (아코디언 섹션):

   | 도구 | 설명 | 상태 |
   |------|------|------|
   | 📰 **뉴스레터 큐레이션** | 딜 + 뉴스 선택 → **매거진 에디터**로 자동 이동 | ✅ 활성 (에디터 연동 완료) |
   | 🤖 **AI 코멘트 어시스턴트** | 짧은 아이디어 입력 → 전문적인 브로커 톤 코멘트 생성 | ✅ **실제 작동** (버그 수정 완료) |
   | 🔗 **화이트라벨 공유** | 커스텀 슬러그 → DB에 저장 + **카톡 공유 활성화** | ✅ 활성 (슬러그 DB 연동 + 카톡 전송) |
   | 📋 **매각준비도 발송** | 자산 선택 → 진단 점수 확인 → 코멘트 + **카톡 전송** | ✅ **신규** |
   | 🤝 **공동중개 제안** | 수수료 분배 설정 + 타겟 오디언스 선택 | 데모 |
   | 🔥 **리드 스코어링** | 매칭 데이터 기반 매수자 관심도 점수 | 활성 |
   | 👷 **벤더 리뷰** | 협력 전문업체 평가 | 샘플 |

3. **핵심 데모 ①: AI 코멘트 어시스턴트**
   - 입력란에 "성수 근생 시장 전망"을 입력 → "✨ AI 말투 생성" 클릭
   - AI가 전문적인 브로커 톤의 시장 분석 코멘트를 생성
   - **복사** 버튼으로 카카오톡 단체방에 즉시 활용

4. **핵심 데모 ②: 매각준비도 발송** **(신규)**
   - 자산(건물) 드롭다운 선택 → 최근 진단 점수 자동 로드 (예: 65/100점)
   - "소유주 전달용 코멘트" 텍스트 입력 (예: "임대차 현황 요약표가 준비되어 블라인드 티저 생성이 가능합니다.")
   - **"💬 카톡으로 전송"** 버튼 클릭 → Kakao SDK `sendDefault`로 즉시 공유
   - 또는 **"📋 링크 복사"** 버튼으로 SMS/이메일용 딥링크 복사

5. **핵심 데모 ③: 화이트라벨 카톡 공유** **(활성화됨)**
   - 고유 주소(Slug) 입력 + 테마 색상 선택 → "🔗 내 매거진 링크 생성 및 저장"
   - 생성된 링크 하단의 **"💬 카톡 공유"** 버튼이 이제 **실제로 동작** — Kakao SDK로 매거진 피드 카드 전송

#### 🎤 멘트 예시
> "스튜디오에서 두 가지를 기억하세요. 첫째, **AI 코멘트** — '성수 근생 전망'만 쓰면 전문가 수준의 카톡 포스팅이 나옵니다. 둘째, **매각준비도 발송** — 건물주에게 '당신 건물의 매각 준비 상태'를 코멘트와 함께 카톡으로 보내세요. 건물주가 진단 링크를 열면 여러분에게 자연스럽게 연결됩니다."

#### 📊 기대 가치
| 기존 | DealCard 콘텐츠 스튜디오 |
|------|-------------------------|
| 카페 단문 글 수동 작성 | AI가 **전문가 톤** 자동 생성 |
| 뉴스레터 직접 구성 | 딜 + 뉴스 선택 → **매거진 에디터 자동 연동** |
| 공동중개 제안 전화로 협의 | **수수료 분배 시스템** 자동화 |
| 건물주에게 별도 전화 연락 | **매각준비도 카톡 발송** — 코멘트 첨부 |
| 매거진 링크 수동 복사 | **화이트라벨 카톡 공유 버튼** 한 번 클릭 |

## ACT 5. CRE 시장 펄스 — 실시간 시장 심리 (2분)

### 📍 화면: `https://www.credeal.net/pulse`

1. **시장 감성 투표** — "현재 시장이 강세? 약세? 관망?" 실시간 투표

#### 🎤 멘트 예시

- **객관적 근거 확보**: "제 판단이 아니라 업계 전문가 82%가 강세 전망입니다"

- **시장 타이밍 포착**: 감성 전환점에서 선제적 고객 연락
