## 1. 시스템 소개

### 1.1 JS 1분 딜카드란?

**JS 1분 딜카드**는 상업용 부동산(CRE) 중개인을 위한 **AI 기반 딜 어시스턴트**입니다.

카카오톡으로 받은 매물 메모 하나만 붙여넣으면, AI가 자동으로:

- ✅ **블라인드 딜카드**를 생성합니다 (민감정보 자동 차단)
- ✅ **매수자 조건**을 구조화합니다 (예산·지역·목적 AI 정리)
- ✅ **매물-매수자 매칭**을 수행합니다 (3단계 AI 매칭 엔진)
- ✅ **고객 CRM**을 관리합니다 (매도자/매수자 통합 관리)
- ✅ **딜 파이프라인**을 추적합니다 (8단계 거래 라이프사이클)

### 1.2 핵심 가치

| 가치 | 설명 |
|------|------|
| ⚡ **1분 딜카드** | 카톡 메모를 붙여넣기만 하면 블라인드 딜카드가 즉시 생성됩니다 |
| 🔒 **정보 보호** | 정확한 주소, 임차인명, 호실별 임대료, 매도자 사정이 자동으로 차단됩니다 |
| 🎯 **AI 매칭** | 3단계 매칭 엔진이 S/A/B/C 등급으로 최적의 매수자를 추천합니다 |
| 📊 **딜 관리** | 메모 입력부터 계약 완료까지 8단계 파이프라인을 한눈에 관리합니다 |

### 1.3 화면 구성 개요

시스템은 **5개 핵심 탭**으로 구성됩니다. 화면 하단 네비게이션 바에서 언제든 이동할 수 있습니다.

| 탭 | 아이콘 | 기능 |
|----|--------|------|
| 홈 | 🏠 | 코크핏 대시보드 — KPI, 빠른 작업, 파이프라인 현황 |
| 고객 | 👥 | 고객 CRM — 매도자/매수자 통합 관리 |
| 매물 | 🏢 | 매물(건물) 목록 및 상세 |
| 매칭 | 🎯 | 매칭 센터 — S/A/B/C 등급별 매칭 결과 |
| 파이프 | 📊 | 딜 파이프라인 — 8단계 칸반 보드 |

## 2. 시작하기 — 회원가입 & 로그인

### 2.1 회원가입

1. 브라우저에서 시스템에 접속합니다.
2. **`회원가입`** 버튼을 클릭합니다.
3. 이메일과 비밀번호를 입력하고 가입합니다.
4. 가입 완료 후 자동으로 로그인됩니다.

> **💡 Tip**: 가입 시 역할이 자동으로 `broker`(중개인)로 설정됩니다.

### 2.2 로그인

1. `/login` 페이지에 접속합니다.
2. 가입 시 사용한 이메일과 비밀번호를 입력합니다.
3. **`로그인`** 버튼을 클릭합니다.
4. 로그인 성공 시 **코크핏 대시보드(`/broker`)**로 이동합니다.

### 2.3 로그아웃

1. 하단 네비게이션 → `홈` → 오른쪽 상단 **`👤`** 프로필 아이콘 클릭
2. 프로필 페이지 하단의 **`로그아웃`** 버튼 클릭

## 3. 코크핏 대시보드 (홈)

### 3.1 접근 방법

- 하단 네비게이션 바에서 **🏠 홈** 탭을 클릭
- URL: `/broker`

### 3.2 화면 구성

코크핏 대시보드는 중개인의 현재 비즈니스 현황을 한눈에 보여주는 **6개 영역**으로 구성됩니다.

#### ① 인사말 & 프로필

화면 최상단에 시간대별 인사말("좋은 아침이에요", "좋은 오후예요", "수고하셨어요")이 표시됩니다. 오른쪽 👤 아이콘을 클릭하면 프로필 관리 페이지로 이동합니다.

#### ② KPI 카드 (4개)

| 카드 | 내용 | 클릭 시 이동 |
|------|------|------------|
| **매물** | 등록한 매물(건물) 총 수 | 매물 목록 (`/broker/buildings`) |
| **고객** | 등록한 고객(매도자/매수자) 수 | 고객 목록 (`/broker/clients`) |
| **S/A 매칭** | S급 + A급 매칭 건수 | 매칭 센터 (`/broker/matching`) |
| **매수자** | 등록한 매수자 의향서 수 | 매수자 목록 (`/broker/buyer-intents`) |

#### ③ 빠른 작업

| 작업 | 설명 |
|------|------|
| **📋 카톡 매물 → 1분 딜카드** | 매물 메모를 붙여넣으면 AI가 블라인드 딜카드 생성 |
| **🎯 매수자 등록** | 매수자 상담 내용을 AI가 구조화 |
| **👥 고객 등록** | 매도자/매수자를 고객으로 CRM에 등록 |

#### ④ 파이프라인 미니 현황

현재 진행 중인 딜을 **초안 / 활성 / 매칭됨** 3단계로 요약 표시합니다. 클릭하면 전체 파이프라인 칸반 보드로 이동합니다.

#### ⑤ 최근 딜카드

가장 최근에 만든 딜카드 5건을 보여줍니다. 각 카드에는 **권역, 자산 유형, 가격대, 상태(활성/초안), 매칭 수**가 표시됩니다.

#### ⑥ 등록된 매수자

최근 등록한 매수자 의향서 3건을 보여줍니다.

## 4. 1분 딜카드 생성

### 4.1 접근 방법

- 코크핏 대시보드 → **📋 카톡 매물 → 1분 딜카드** 클릭
- URL: `/broker/deal-card/new`

### 4.2 사용 방법

#### Step 1: 매물 메모 붙여넣기

카카오톡으로 받은 매물 설명을 그대로 텍스트 입력란에 붙여넣습니다.

```
예시:
성수동 쪽 80억대 근생 건물. 정확한 주소는 아직 비공개로 해주세요.
1층은 F&B로 쓰기 괜찮고 상층은 사무실 가능.
일부 임대 중이고 일부 공실 가능성 있음.
매도자는 너무 공개되는 걸 싫어해서 임차인명이나 월세 세부는 빼고
먼저 사옥 수요자나 장기보유형한테 반응 보고 싶어함.
```

> **💡 Tip**: `예시 메모 사용` 버튼을 클릭하면 샘플 메모가 자동으로 입력됩니다. 처음 사용할 때 기능을 이해하는 데 도움이 됩니다.

#### Step 2: 딜카드 생성

**`1분 딜카드 만들기`** 버튼을 클릭합니다. AI가 5단계 과정을 수행합니다:

| 단계 | 처리 내용 |
|------|----------|
| 1 | 메모에서 매물 정보 추출 |
| 2 | 건물 기본 신호 생성 |
| 3 | 민감정보 식별 및 숨김 처리 |
| 4 | 블라인드 딜카드 생성 |
| 5 | 카카오 문구 준비 |

각 단계의 진행 상황이 화면에 실시간으로 표시됩니다. 전체 과정은 약 **10~30초**가 소요됩니다.

#### Step 3: 결과 확인

생성 완료 후 자동으로 **딜카드 상세 페이지**로 이동합니다.

### 4.3 자동 숨김 정보

AI가 자동으로 차단하는 민감 정보:

| 숨김 항목 | 설명 |
|----------|------|
| 정확한 주소 | 도로명/지번 주소가 권역(예: "성수동 일대")으로 대체됩니다 |
| 임차인명 | 개별 임차인의 실명이 모두 제거됩니다 |
| 호실별 임대료 | 호실별 세부 임대 조건이 제거됩니다 |
| 매도자 사정 | 매도 이유, 급매 여부 등이 제거됩니다 |
| 협상 관련 메모 | 가격 협의, 할인 등 내부 메모가 제거됩니다 |

### 4.4 주의사항

> ⚠️ **메모 최소 길이**: 5자 이상이어야 딜카드 생성 버튼이 활성화됩니다.  
> ⚠️ **AI 초안**: 생성된 딜카드는 'AI 초안' 상태이며, 필요 시 수동으로 수정할 수 있습니다.

## 5. 딜카드 상세 & 활용

### 5.1 접근 방법

- 코크핏 대시보드 → 최근 딜카드 항목 클릭
- 매물 목록 → 항목 클릭
- URL: `/broker/deal-card/[id]`

### 5.2 화면 구성

딜카드 상세 페이지는 다음 영역으로 구성됩니다:

#### ① 파이프라인 진행 상태

현재 딜의 파이프라인 단계가 시각적으로 표시됩니다.

```
메모 → 딜카드 → Gate → IM → 미팅 → LOI → 계약 → 완료
```

#### ② 건물 신호 요약

AI가 추출한 건물의 핵심 정보가 4칸 그리드로 표시됩니다:

| 항목 | 예시 |
|------|------|
| 권역 | 성수동 일대 |
| 자산 유형 | 근생 건물 |
| 가격대 | 80억대 |
| 현재 사용 | F&B + 사무실 |

추가로 **적합 매수자**, **확인 필요** 사항이 하단에 표시됩니다.

#### ③ 숨긴 정보

자동으로 차단된 민감 정보 항목이 🔒 아이콘과 함께 노란색 알림으로 표시됩니다.

#### ④ 블라인드 티저 미리보기

매수자에게 전달할 블라인드 딜카드의 최종 미리보기입니다. 포함 정보:

- **제목**: 자동 생성된 딜카드 제목
- **짧은 요약**: 1~2문장 핵심 요약
- **딜 포인트**: 이 매물의 투자 매력 포인트 (✅ 초록색)
- **주의사항**: 확인이 필요한 사항 (⚠️ 노란색)
- **숨김 안내**: 비공개 처리된 정보 안내 문구
- **게이트 메시지**: 상세 정보 열람을 유도하는 메시지

#### ⑤ 카톡 문구

카카오톡으로 바로 전달할 수 있는 문구가 자동 생성됩니다.

- **`카톡 문구 복사`** 버튼을 클릭하면 클립보드에 복사됩니다.
- 카카오톡에 바로 붙여넣기하여 매수자에게 전달할 수 있습니다.

#### ⑥ 매칭된 매수자

등록된 매수자 중 이 매물에 적합한 매수자가 S/A/B/C 등급으로 표시됩니다.

#### ⑦ 가격 예측

AI 기반 거래 가격 예측 결과가 표시됩니다.

### 5.3 하단 액션 바

화면 하단에 고정된 액션 바에서 다음 작업을 수행할 수 있습니다:

| 버튼 | 기능 |
|------|------|
| **💬 카톡 문구 복사** | 블라인드 티저 카톡 문구를 클립보드에 복사 |
| **📄 Full IM 만들기** | 전체 투자 설명서(IM)로 확장 |
| **📱 모바일 IM 만들기** | 모바일 최적화 IM 생성 |
| **🎯 매수자 전체 보기** | 등록된 매수자 목록으로 이동 |
| **📊 건물주 리포트** | 건물주 대상 상세 리포트 |
| **✨ 전문가 코멘트 받기** | 전문가의 3줄 의견 요청 |

### 5.4 Gate 요청

딜카드 하단에 **Gate 요청 폼**이 있습니다. 매수자가 상세 정보 열람을 요청할 때 사용합니다.

- **G1**: 기본 정보 공개
- **G2**: 상세 정보 공개
- **G3**: 전체 정보 공개

각 단계별로 공개되는 정보의 범위가 다릅니다.

## 6. 매수자 조건 등록

### 6.1 접근 방법

- 코크핏 대시보드 → **🎯 매수자 등록** 클릭
- 하단 네비게이션 → 홈 → 빠른 작업
- URL: `/broker/buyer-intents/new`

### 6.2 사용 방법

#### Step 1: 매수자 상담 메모 붙여넣기

매수자와 상담한 내용을 텍스트 입력란에 붙여넣습니다.

```
예시:
김대표님 조건. 예산은 50억에서 80억 정도.
성수나 강남 쪽 선호. 사옥으로 일부 쓰고 나머지는 임대수익 있으면 좋겠다고 함.
주차는 꼭 필요하고, 너무 노후된 건물은 부담스러워함.
대출은 50% 정도까지는 생각하지만 확정 아님.
기존 임차인 만기가 언제인지 중요하게 봄.
```

#### Step 2: AI 정규화

**`조건 정리하기`** 버튼을 클릭합니다. AI가 4단계 과정을 수행합니다:

| 단계 | 처리 내용 |
|------|----------|
| 1 | 매수자 조건 추출 |
| 2 | 예산·지역·목적 구조화 |
| 3 | 필수조건·우대조건 정리 |
| 4 | 추가 확인 질문 생성 |

#### Step 3: 결과 확인

생성 완료 후 자동으로 매수자 의향서 상세 페이지로 이동합니다. AI가 구조화한 정보:

| 항목 | 설명 |
|------|------|
| 매수자 유형 | 개인/법인/기관 등 |
| 예산 범위 | 최소~최대 금액 |
| 선호 지역 | 선호하는 권역 목록 |
| 선호 자산 유형 | 관심 있는 건물 유형 |
| 매수 목적 | 임대수익/사옥/증여/개발/갭투자 |
| 리스크 허용도 | 보수적/중간/적극적 |

> **🔒 프라이버시**: 매수자 실명과 연락처는 공유 문서에 포함되지 않습니다.

## 7. 매수자 의향서 관리

### 7.1 접근 방법

- 하단 네비게이션 → 홈 → 등록된 매수자 → **전체 보기**
- URL: `/broker/buyer-intents`

### 7.2 목록 페이지

등록된 매수자 의향서가 카드 형태로 표시됩니다. 각 카드에는:

| 정보 | 설명 |
|------|------|
| 매수자 유형 | 법인/개인 등 |
| 매수 목적 | 아이콘(💰/🏛️/🎁/🏗️/📈)과 함께 표시 |
| 예산 | 예산 범위 |
| 리스크 | 보수적/중간/적극적 |
| 선호 지역 | 선호 지역 목록 |
| 선호 자산 | 선호 자산 유형 |
| 매칭 건수 | 해당 매수자에 매칭된 매물 수 |
| 등록일 | 의향서 등록 날짜 |

### 7.3 상세 페이지

카드를 클릭하면 매수자 의향서 상세 페이지로 이동합니다. 이 페이지에서는:

- AI가 정리한 **구조화된 조건**을 확인할 수 있습니다
- 조건을 **수정**할 수 있습니다
- 이 매수자에 **매칭된 매물 목록**을 확인할 수 있습니다
- **추가 확인 질문**을 볼 수 있습니다 (AI가 생성한 후속 질문)

### 7.4 신규 등록

목록 페이지 상단 또는 하단의 **`+ 추가`** / **`🎯 매수자 조건 새로 등록하기`** 버튼을 클릭하면 신규 매수자 등록 페이지로 이동합니다.

## 8. 매물(건물) 관리

### 8.1 접근 방법

- 하단 네비게이션 → **🏢 매물** 탭
- URL: `/broker/buildings`

### 8.2 건물 목록

등록된 매물 목록이 표시됩니다. 각 항목에는 **권역, 자산 유형, 가격대, 상태, 매칭 수**가 표시됩니다.

### 8.3 건물 상세

건물 항목을 클릭하면 상세 페이지로 이동합니다. 여기에서 확인할 수 있는 정보:

- 건물 기본 정보 (권역, 자산유형, 가격대 등)
- 파이프라인 진행 상태
- 매칭된 매수자 목록
- 건물 관련 문서 (딜카드, IM 등)

### 8.4 새 매물 등록

새 매물은 **1분 딜카드 만들기**를 통해 등록합니다. 카톡 메모를 입력하면 건물 정보가 자동으로 생성되고 매물로 등록됩니다.

## 9. 프로필 관리

### 9.1 접근 방법

- 코크핏 대시보드 → 오른쪽 상단 **👤** 아이콘 클릭
- URL: `/broker/profile`

### 9.2 관리 항목

| 항목 | 설명 |
|------|------|
| **이름** | 중개인 표시 이름 |
| **연락처** | 전화번호 |
| **소속/회사** | 중개법인명 |
| **전문 지역** | 주로 활동하는 지역 (복수 선택 가능) |
| **전문 자산 유형** | 주로 다루는 자산 유형 (복수 선택 가능) |
| **자기소개** | 경력, 전문 분야, 고객 메시지 (최대 500자) |

### 9.3 전문 지역 선택 가능 항목

```
강남구, 서초구, 송파구, 마포구, 용산구, 영등포구, 여의도, 종로구, 중구,
강북구, 노원구, 도봉구, 강동구, 광진구, 성동구, 기타 서울,
경기 분당, 경기 판교, 경기 수원, 인천
```

### 9.4 전문 자산 유형 선택 가능 항목

```
상가, 오피스빌딩, 꼬마빌딩, 지식산업센터, 물류센터,
호텔/숙박, 주유소, 근린상가, 복합용도, 토지
```

### 9.5 저장

모든 항목을 입력/수정한 후 **`프로필 저장`** 버튼을 클릭합니다. "✓ 프로필이 저장되었습니다." 메시지가 표시되면 저장이 완료된 것입니다.
