## 📊 네이버·직방 vs 크리딜 — 근본적 차이

### 네이버 부동산 / 직방

```
건물 등록 → 매물 목록에 노출 → 매수자가 알아서 검색 → 전화 옴
         ↓
    "수동 대기형"
    매수자가 찾아오기를 기다린다
```

| 항목 | 네이버/직방 | 크리딜 |
|:---|:---|:---|
| **핵심 모델** | 매물 검색 포털 (부동산114 스타일) | 딜 매칭 엔진 (투자은행 스타일) |
| **매칭 방식** | 지역+가격 필터 → 목록 나열 | **AI가 매수자 목적을 분석하여 매칭** |
| **브로커 역할** | 매물 사진+설명 올리는 사람 | **딜을 설계하고 매칭을 받는 전문가** |
| **매수자 이해** | 없음 (검색 키워드만 봄) | **매수 목적·예산·성향을 AI가 프로파일링** |
| **데이터 축적** | 내 매물 조회수만 | **매칭 성사/실패 패턴 → 점점 정확해짐** |
| **컴플라이언스** | 없음 (알아서 조심) | **AI가 위험 문구 자동 제거** |

### 해외 플랫폼 비교

| 플랫폼 | 핵심 기능 | 크리딜 vs 차이 |
|:---|:---|:---|
| **CoStar** (미국 1위) | 시장 데이터 + 매물 DB | 연 ₩3,000만원+ 구독료. 매칭 없음 |
| **LoopNet** | 매물 목록 포털 | 네이버와 동일 모델. 매수자 분석 없음 |
| **Crexi** | 매물 마켓플레이스 | 단순 필터 매칭. 목적별 가중치 없음 |
| **RealPage** | 임대 관리 SaaS | 매매 딜 매칭 기능 없음 |
| **크리딜** | **딜 매칭 AI OS** | **목적별 가중치 + 딜 스토리 점수 + 매수자 페르소나** |

> **세계 어디에도 "매수자의 매입 목적에 따라 매칭 기준을 바꾸는" 시스템은 없습니다.**

## ⚡ 크리딜 AI 매칭 — 3가지 핵심 무기

### 🎯 무기 1: "왜 사려는지"에 따라 매칭이 달라진다

```
같은 "테헤란로 80억 오피스"를 보는 매수자 3명:

                    네이버/직방:           크리딜:
                    ━━━━━━━━━━           ━━━━━━━━
매수자 A (사옥용)    "80억, 강남" → 목록    AI가 "사옥" 감지
                    에 나옴                → 입지 35%, 재무 25%
                                          → "역세권+독립동선" 중점 매칭
                                          → A등급 ✅

매수자 B (투자용)    동일 검색결과           AI가 "투자" 감지
                                          → 재무 40%, 시장 20%
                                          → "Cap Rate 4.8%" 중점 매칭
                                          → S등급 ✅✅

매수자 C (증여용)    동일 검색결과           AI가 "증여" 감지
                                          → 세무 50%, 시장 10%
                                          → "감정가갭+증여세" 분석
                                          → B등급 (세무 불리)

🔑 네이버/직방: 3명 모두 같은 목록을 봅니다
🔑 크리딜:     3명에게 완전히 다른 매칭 등급과 이유를 제시합니다
```

> **실무에서 "사옥 찾는 법인"과 "투자 찾는 개인"에게 같은 방식으로 매물을 소개하십니까? 아닙니다. 크리딜 AI가 이것을 자동으로 해드립니다.**

---

### 📖 무기 2: "이 매물이 얼마나 흥미로운 이야기를 가지고 있는가"

```
Deal Curiosity Score — 딜 스토리 점수 (0~100)

높은 점수 (잘 팔리는 매물):
┌─────────────────────────────────────────────┐
│ 88점 "용도변경 가능 종로 근생"               │
│  → 스토리: "1종→2종 변경 시 4층 증축 가능"   │
│  → 매수자에게 할 말이 많다                   │
│  → 미팅 요청률 3배↑                         │
└─────────────────────────────────────────────┘

낮은 점수 (소개가 어려운 매물):
┌─────────────────────────────────────────────┐
│ 35점 "평범한 역삼 오피스 전층 임대"           │
│  → 스토리: "임대 중... 그게 전부"            │
│  → AI가 "누락 자료 5건" 알려줌               │
│  → 자료 보완하면 점수↑                       │
└─────────────────────────────────────────────┘

🔑 네이버/직방: 모든 매물이 "사진 + 가격 + 면적"으로 동일하게 보임
🔑 크리딜:     "이 매물은 이런 이야기로 팔아야 합니다"를 AI가 알려줌
```

## 🆕 신규 기능: AI 매수자 페르소나 — "이 매물은 누가 사야 할까?"

> **매물을 등록하면, AI가 "이상적 매수자 3명의 프로필"을 자동으로 만들어 드립니다.**

```
테헤란로 80억 오피스 등록 시:

AI가 자동으로 3가지 이상적 매수자 페르소나를 도출:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

👤 페르소나 A: "IT 중견기업 사옥 이전형"
   유형: 법인 (매출 500억~2000억 IT/게임)
   예산: 70~100억
   동기: 판교/강남 사옥 이전, 직원 접근성 중시
   핵심 니즈: 역세권 + 단독사옥 + 주차 20대+
   어디서 찾나: 판교 IT기업 이전 뉴스, 법인등기 변경 모니터링
   접근법: "직원 통근 만족도 + 브랜드 가치 상승" 어필

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

👤 페르소나 B: "자산가 절세형 투자자"
   유형: 개인 (순자산 100억+, 50대 남성)
   예산: 80~120억
   동기: 부동산 포트폴리오 분산, 증여 대비
   핵심 니즈: Cap Rate 4%+ + 감정가 갭 + 안정 임차인
   어디서 찾나: 세무사/회계사 네트워크, 프라이빗뱅킹(PB)
   접근법: "감정가 대비 저평가 + 증여 시 절세 효과" 어필

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

👤 페르소나 C: "외국계 기업 한국 진출형"
   유형: 외국법인 (미국/일본 IT/바이오)
   예산: 60~90억 (본사 승인 기반)
   동기: 한국 지사 설립, GBD 입지 필수
   핵심 니즈: 국제학교 인접 + 영어 가능 관리 + A급 내장
   어디서 찾나: KOTRA, 외국상공회의소, 외국계 로펌
   접근법: "GBD 핵심 입지 + 즉시 입주 가능" 어필

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 각 페르소나별 "어디서 찾나" + "어떻게 접근하나"까지 AI가 제시
→ 막연히 "누구 없나?" 대신 "이런 사람을 이렇게 찾아라"
```

## 💡 실무 시나리오: 크리딜 하루

```
오전 9시 ─── 출근
  📱 모닝 인텔리전스 카톡 도착
     "어제 성수 근생 55억 매물에 A등급 매수자 1명 새로 매칭됨"
     "테헤란로 오피스 딜 성사확률 72%→78% 상승 (매수자 미팅 효과)"

오전 10시 ─── 신규 매물 접수
  📝 카톡 메모 → 크리딜에 붙여넣기 (30초)
     → 딜카드 자동 생성
     → AI 매수자 페르소나 3명 자동 도출
     → "페르소나 A는 PB 네트워크에서 찾으세요" 안내

오전 11시 ─── 매수자 미팅 준비
  🎧 모바일 IM 음성 브리핑을 들으며 핵심 정리
     → 매수자에게 IM 링크 카톡 전송

오후 2시 ─── 매수자 반응 확인
  📊 "매수자 김XX가 IM 7개 섹션 중 5개 열람 (체류 4분 32초)"
     → "수익분석 섹션에서 가장 오래 머물렀습니다"
     → AI: "재무에 관심이 높습니다. Cap Rate 자료 추가 전송을 권합니다"

오후 5시 ─── 퇴근 전
  📋 크리딜 대시보드에서 파이프라인 확인
     → 진행 중 딜 5건, 성사확률 Top 3 확인
     → AI: "성수 근생은 매수자 B가 이번 주 미팅 가능합니다"
```

## 🆚 정리: "왜 크리딜인가?"

```
┌──────────────────────────────────────────────────────────┐
│                                                          │
│   네이버/직방:  "매물을 올리고 기다린다"                    │
│                 → 수동적. 브로커가 할 일이 없다             │
│                                                          │
│   크리딜:       "AI가 매물을 분석하고,                     │
│                  매수자를 찾아주고,                        │
│                  전략을 알려준다"                          │
│                 → 능동적. 브로커가 전문가가 된다             │
│                                                          │
│   한마디로:                                               │
│   "네이버는 전단지판,                                     │
│    크리딜은 전담 애널리스트"                                │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 브로커님께 드리는 약속

| # | 약속 | 방법 |
|:---:|:---|:---|
| 1 | **매물 등록 30초** | 카톡 메모 붙여넣기 → AI 자동 딜카드 |

| 5 | **컴플라이언스 자동** | 위험 문구 AI 제거 → 법적 안전 |
| 6 | **성과 측정** | 열람·매칭·미팅 데이터 → 내 실적 입증 |
