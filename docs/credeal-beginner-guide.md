## 1. 접속하기

### 주소
브라우저에서 아래 URL로 접속합니다.

```
https://credeal.net
```

> 💡 **모바일 최적화**: Credeal은 모바일(스마트폰) 환경에 최적화된 웹 앱입니다.  
> iPhone의 Safari 또는 Android의 Chrome에서 접속하면 가장 편리합니다.

### 홈 화면에 추가하기 (선택)
모바일에서 앱처럼 빠르게 접속하려면:
- **iPhone**: Safari에서 접속 → 하단 공유 버튼(□↑) → **"홈 화면에 추가"**
- **Android**: Chrome에서 접속 → 우측 상단 메뉴(⋮) → **"홈 화면에 추가"**

홈 화면에 "JS 딜카드" 아이콘이 추가되며, 앱처럼 바로 실행할 수 있습니다.

## 2. 회원가입

### 접속 경로
```
https://credeal.net/signup
```
또는 로그인 화면 하단의 **"아직 계정이 없으신가요? → 회원가입"** 링크를 탭합니다.

### 입력 항목

| 항목 | 설명 | 예시 |
|------|------|------|
| **이름 / 닉네임** | 2~30자, 실명 또는 닉네임 | 홍길동 |
| **이메일** | 로그인 시 사용할 이메일 | hong@example.com |
| **비밀번호** | 8자 이상 | ●●●●●●●● |
| **계정 유형** | 아래 두 가지 중 택 1 | — |

### 계정 유형 선택

- 👤 **일반 사용자** — 매수자, 건물주 등 일반 이용자
- 🏢 **공인중개사** — 부동산 전문가 (중개사, 법무사, 세무사 등)

> 💡 **중요**: 중개인이시라면 반드시 **🏢 공인중개사**를 선택하세요. 딜카드, 매거진, 매칭 등 전문가 전용 기능이 활성화됩니다.

### 가입 완료
**"가입하기"** 버튼을 누르면 계정이 즉시 생성되며, 자동으로 중개인 대시보드(`/broker`)로 이동합니다.

> ⚠️ 이미 가입된 이메일은 **"이미 사용 중인 이메일입니다."** 오류가 표시됩니다. 이 경우 로그인으로 이동하세요.

## 3. 로그인

### 접속 경로
```
https://credeal.net/login
```

### 로그인 방법
1. 가입 시 사용한 **이메일** 입력
2. **비밀번호** 입력
3. **"로그인"** 버튼 탭

로그인 성공 시 중개인 대시보드(`/broker`)로 자동 이동합니다.

> ⚠️ 이메일 또는 비밀번호가 틀리면 **"이메일 또는 비밀번호가 올바르지 않습니다."** 메시지가 표시됩니다.

### 로그아웃
대시보드 우측 상단의 **"로그아웃"** 버튼을 탭하면 로그아웃됩니다.

## 4. 온보딩 제대로 하기

### 온보딩이란?
첫 로그인 후 자동으로 시작되는 **초기 설정 과정**입니다.  
AI가 프로필 사진을 분석하여 **Vibe 명함**(디지털 명함)을 자동 생성하고, 전문 분야와 활동 지역을 설정합니다.

### 접속 경로
```
https://credeal.net/onboarding
```
> 이미 온보딩을 완료했다면 자동으로 대시보드로 이동합니다. 온보딩을 아직 하지 않은 신규 가입자는 자동으로 이 페이지로 안내됩니다.

### 진행 순서 (7~8단계)

화면 상단의 **진행 바(●●●●●)**로 현재 진행 상황을 확인할 수 있습니다.

---

#### ① 역할 선택

두 가지 역할 카드 중 하나를 선택합니다:

- 🏢 **부동산 전문가** — 공인중개사, 법무사, 세무사, 감정평가사
- 🏠 **건물주 / 투자자** — 건물 소유주, 매수 관심자, 개발사

> 💡 **JS 중개인이시라면 "부동산 전문가"를 선택**하세요. 전문가 경로는 빌딩 레이더, 딜카드 생성 등 추가 기능을 체험할 수 있습니다.

---

#### ② 프로필 사진 업로드

**얼굴이 보이는 프로필 사진**을 업로드합니다.

> ⚠️ **얼굴 사진 필수!** AI Vibe 분석은 사진 속 인상을 기반으로 신뢰도, 성향 프로필을 생성합니다. 풍경 사진이나 로고 등은 분석 품질이 크게 떨어집니다.

**추천 사진 조건:**
- 정면 또는 약간 비스듬한 상반신 사진
- 밝은 조명, 자연스러운 표정
- 정장 또는 깔끔한 복장
- 최소 300×300px 이상

---

#### ③ AI 분석 중

사진 업로드 후 **약 10~20초** 동안 AI가 분석합니다.  
화면에 프로필 사진 미리보기와 분석 진행 바가 표시됩니다. 잠시 기다려 주세요.

---

#### ④ 분석 결과 확인

AI가 생성한 **Vibe 프로필**이 공개됩니다:

- **VTI 유형**: MBTI처럼 중개 스타일을 표현하는 아이콘과 레이블 (예: 🏛️ 자산전략가)
- **신뢰 점수 / 밸런스 점수**: 사진에서 분석된 인상 기반 점수
- **Before/After**: 분석 전후 점수 비교

여기서 **"공유하기"** 버튼을 탭하면 분석 결과를 카카오톡으로 바로 공유할 수도 있습니다.

**"계속"** 을 탭하여 다음 단계로 이동합니다.

---

#### ⑤ 상세 정보 입력 (명함 완성)

이 단계는 3개의 하위 단계로 나뉩니다:

**5-1. 기본 정보**
- **이름** (필수): 명함에 표시될 이름
- **연락처** (선택): 전화번호

**5-2. 전문 분야 선택**
아래 중 하나 이상을 탭합니다:
- 꼬마빌딩 · 오피스임대 · 리테일 · 산업용 · 변호사 · 세무사 · 법무사 · 기타

**5-3. 활동 지역 선택**
지역을 탭하면 자동으로 다음 단계로 넘어갑니다:
- 성수/성동 · 강남/서초 · 여의도/마포 · CBD · 판교 · 기타

> 💡 "나중에 하기"를 탭하면 건너뛸 수 있지만, **가능하면 여기서 전문 분야와 지역을 선택하는 것을 추천**합니다. 매칭 알고리즘이 이 정보를 활용합니다.

---

#### ⑥ 빌딩 레이더 (전문가 경로만)

건물 주소를 입력하여 **AI 건물 분석**을 체험합니다.

- 주소 입력 후 "분석" 버튼 탭
- AI가 해당 건물의 시장 시그널, 거래 현황 등을 분석

> 💡 건너뛰기("나중에 하기")를 탭해도 됩니다. 대시보드에서 언제든 이용 가능합니다.

---

#### ⑦ 첫 번째 딜카드 만들기 (전문가 경로만)

카카오톡에서 보내던 매물 소개 문구를 붙여넣으면 **30초 만에 블라인드 딜카드**가 생성됩니다.

> 💡 건너뛰기 가능합니다. 대시보드의 "30초 딜카드" 버튼으로 언제든 만들 수 있습니다.

---

#### ⑧ 온보딩 완료! 🎉

축하 화면이 나타납니다! 달성한 항목을 체크리스트로 보여줍니다:
- 🎯 AI Vibe 프로필 명함
- 🏢 빌딩 레이더 첫 조회 (진행한 경우)
- 📋 블라인드 딜카드 1개 (진행한 경우)

**Vibe 명함 미리보기**가 하단에 표시됩니다. 여기서:

- **📱 카톡으로 공유하기**: 명함 링크를 카카오톡으로 바로 공유
- **대시보드로 이동**: 중개인 코크핏(대시보드)으로 이동

## 5. 프로필 관리

### 접속 경로
```
https://credeal.net/broker/profile
```
또는 대시보드 우측 상단의 **👤 프로필 아이콘**을 탭합니다.

### 프로필 완성도 게이지

상단에 **0~100% 프로필 완성도** 바가 표시됩니다. 항목별 기여도:

| 항목 | 기여도 | 비고 |
|------|--------|------|
| 이름 입력 | +5% | — |
| 연락처 입력 | +5% | — |
| 자격증 번호 | +15% | 공인중개사 자격증 |
| 개업 연도 | +10% | 경력 자동 계산 |
| 전문 지역 3개 이상 | +10% | — |
| 전문 자산 유형 2개 이상 | +10% | — |
| 거래 실적 입력 | +15% | — |
| 자기소개 100자 이상 | +10% | — |
| 소셜 링크 1개 이상 | +10% | — |
| 거래 금액대 선택 | +10% | — |
| 프로필 사진 등록 | +10% | — |

> 💡 **70% 이상** 달성 시 초록색으로 변하며, Vibe 명함과 매칭 정확도가 높아집니다.

### 수정 가능 항목

#### 기본 정보
- **프로필 사진**: 사진 변경 시 AI가 자동으로 Vibe를 재분석합니다
- **이름**, **연락처**, **소속/회사**, **공유 타이틀(OG Title)**

#### 자격/등록 정보
- 공인중개사 자격증 번호, 중개사무소 등록번호
- 소속 협회, 개업 연도 (경력 연수 자동 계산)

#### 전문 지역
강남구, 서초구, 성수동, 여의도 등 20개 지역 중 다수 선택

#### 전문 자산 유형
상가, 오피스빌딩, 물류센터, 호텔 등 10개 유형 중 다수 선택

#### 거래 실적
- 총 거래 건수, 전문 거래 금액대 (10억 미만 ~ 500억 이상)
- 딜 유형 전문성, 매수자 유형

#### 서비스 정책
- 수수료 정책 (법정수수료 준수 / 별도 협의)
- 상담 방식, 응답 보장 시간, 소통 가능 언어

#### 소셜 링크
카카오채널, 네이버 블로그, 유튜브, LinkedIn URL

#### 자기소개
- 직접 작성 (최대 1,000자)
- **✨ "AI로 자기소개 생성하기"** 버튼을 탭하면 입력된 정보를 기반으로 AI가 전문적인 자기소개를 자동 작성해 줍니다

### 저장하기
모든 수정을 마친 후 하단의 **"프로필 저장"** 버튼을 탭합니다.

## 6. Vibe 명함 관리하기

### 접속 경로
```
https://credeal.net/broker/vibe-card
```
또는 대시보드 퀵 액션 영역의 **"✨ Vibe 명함"** 버튼을 탭합니다.

### Vibe 명함이란?
AI가 프로필 사진을 분석하여 자동으로 생성한 **디지털 명함**입니다. 7차원 성향 벡터, VTI 유형(예: 🏛️ 자산전략가), 신뢰 점수 등이 표시됩니다.

### 관리 페이지 구성

#### 명함 미리보기
상단에 **Vibe 명함 전체 미리보기**가 표시됩니다. 탭하면 공개 페이지가 새 탭에서 열립니다.

#### 4개 액션 버튼 (2×2 그리드)

| 버튼 | 기능 |
|------|------|
| 📷 **사진 변경** | 새 프로필 사진 업로드 → AI 자동 재분석 |
| 🔄 **재분석** | 현재 사진으로 AI Vibe 분석 다시 실행 |
| 📋 **URL 복사** | 공개 명함 URL을 클립보드에 복사 |
| 🔗 **공개 보기** | 공개 명함 페이지를 새 탭에서 열기 |

#### 로고 설정
- **회사 로고** (좌측): 회사/중개사무소 로고 업로드
- **제휴사 로고** (우측): 제휴사 로고 업로드
- 지원 형식: PNG, WebP, SVG (최대 2MB, 투명 배경 PNG 추천)

#### 공개 URL
명함의 공개 URL이 표시됩니다:
```
https://credeal.net/vibe-card/홍길동-abc123
```
이 URL을 고객에게 보내면 누구나 내 명함을 볼 수 있습니다.

### 명함 공유하기

우측 상단의 **공유 아이콘**을 탭하면 공유 옵션이 나타납니다:

| 옵션 | 설명 |
|------|------|
| **URL 복사** | 명함 링크를 클립보드에 복사 |
| **카카오톡 공유** | 카카오톡으로 OG 이미지가 포함된 리치 메시지 전송 |
| **이미지 저장** | 명함을 이미지로 저장 (인쇄) |
| **QR 코드 보기** | QR 코드 표시 → 스캔으로 명함 열기 |

### 공개 명함 페이지에 표시되는 정보
고객이 명함 링크를 열면 아래 정보가 표시됩니다:
- 프로필 사진, 이름, VTI 유형
- 전문 분야, 활동 지역
- 자격증 정보, 경력 연수
- 수수료 정책, 카카오 채널
- 최근 공개한 IM(투자설명서) 목록
- 최신 발행 매거진

## 7. 메모 기능 / 메모함 활용하기

### 핵심 포인트
> 🎯 **Credeal 메모는 단순한 메모장이 아닙니다.** AI가 메모 내용을 분석하여 자동으로 딜카드, 매수 의향서, 일정 등을 생성해 줍니다.

### 메모 입력하기 (유니버설 메모)

대시보드(`/broker`)의 **우측 하단**에 노란색 떠 있는 버튼(**📝 메모 FAB**)이 항상 보입니다.

1. **📝 버튼을 탭**합니다
2. 두 가지 입력 방식 중 하나를 선택합니다:

#### 🎤 음성으로 말하기
- 마이크 아이콘이 빨간색으로 깜빡이면 **녹음 중**입니다
- 한국어로 자연스럽게 말하세요 (실시간 음성 인식)

## 8. 딜카드 기능 활용하기

### 딜카드란?
카카오톡에서 보내던 매물 소개 문구를 붙여넣으면, AI가 자동으로 **민감정보를 숨긴 블라인드 딜카드**를 생성합니다.

### 딜카드 만들기

#### 접속 경로
```
https://credeal.net/broker/deal-card/new
```
또는 대시보드의 **"📝 30초 딜카드"** 퀵 액션 버튼을 탭합니다.

#### 생성 과정

**Step 1. 매물 정보 붙여넣기**
- 큰 텍스트 입력창이 나타납니다
- **평소 카카오톡으로 보내던 매물 소개 문구 그대로** 붙여넣으세요
- 예시 메모 보기: **"예시 메모 사용"** 버튼을 탭하면 샘플 텍스트가 자동 입력됩니다

> 💡 입력 예시:
> ```
> 성수동 카페거리 5층 꼬마빌딩
> 매매가 80억 (감정가 75억)
> 1층 스타벅스, 2~4층 사무실, 5층 공실
> 보증금 총 8억 / 월세 3,200만
> 연 수익률 약 4.8%
> 건물주 해외 이주로 급매
> ```

**Step 2. 사진 첨부 (선택)**
- 건물 사진을 첨부할 수 있습니다 (여러 장 가능)
- 딜카드에 사진이 갤러리로 표시됩니다

**Step 3. "1분 딜카드 만들기" 버튼 탭**

AI가 5단계로 처리합니다 (약 30~60초):
1. ✅ 메모에서 매물 정보 추출 중
2. ✅ 건물 기본 신호 생성 중
3. ✅ 민감정보 식별 및 숨김 처리 중
4. ✅ 블라인드 딜카드 생성 중
5. ✅ 카카오 문구 준비 중

> ⚠️ 입력 내용에 **지역, 자산 유형, 숫자(가격/수익률), 거래 유형** 중 하나라도 빠지면 품질 경고가 표시됩니다. 가능하면 모두 포함시켜 주세요.

**Step 4. 생성 완료!**
세 가지 다음 행동이 제안됩니다:
1. 🟡 **딜카드 확인 및 카톡 공유** → 딜카드 상세 페이지로 이동
2. 📅 **임장 스케줄 설정하기** → 스케줄 관리
3. 📋 **내 딜카드 목록 보기** → 전체 매물 목록

### 딜카드 상세 화면

```
https://credeal.net/broker/deal-card/{딜카드ID}
```

#### 화면 구성

| 섹션 | 내용 |
|------|------|
| **사진 갤러리** | 업로드된 건물 사진 슬라이드 |
| **건물 신호 요약** | 권역, 자산 유형, 가격대, 현재 사용 (2×2 카드) |
| **🔒 숨긴 정보** | AI가 자동으로 숨긴 민감정보 목록 (정확한 주소, 임차인명, 매도자 사정 등) |
| **딜카드 에디터** | 제목, 개요, 핵심 딜 포인트, 카톡 공유 문구 등을 직접 수정 가능 |
| **매칭된 매수자** | AI가 찾은 적합 매수자 목록 |
| **AI 거래 예측** | AI의 거래 가능성 분석 |
| **게이트 요청 관리** | 상세 정보 요청 접수 현황 |

#### 딜카드 에디터 사용법
- 📌 **제목**: 매물의 캐치프레이즈 수정
- 📋 **딜 개요**: 전체 요약 수정
- 🔑 **핵심 딜 포인트**: 항목 추가/삭제 가능
- 💬 **카톡 공유 문구**: 카카오톡으로 보낼 문구 수정
- **"미리보기"**: 실제 공유 시 보이는 화면 미리보기

## 9. 소통 관리함 활용하기

### 접속 경로

```
또는 대시보드 우측 상단의 **🔔 종 아이콘**을 탭합니다.

> 💡 새로운 요청이 있으면 종 아이콘 옆에 **빨간 뱃지 숫자**가 표시됩니다.

### 소통 관리함이란?
딜카드, 모바일 IM, 매거진, Vibe 명함 등에서 발생하는 **고객 반응(열람, 상세 요청)을 한 곳에서 종합 관리**하는 허브입니다.

### 3개 탭 필터

| 탭 | 내용 |
|----|------|
| **전체** | 모든 알림 및 요청을 시간순으로 표시 |
| **상세정보 요청 (N)** | 처리 대기 중인 Gate Request만 모아보기 (건수 뱃지 표시) |
| **열람/반응** | IM, 딜카드, 매거진, 명함 열람 기록 |

### 상세 정보 요청 처리하기

고객이 딜카드 하단의 "상세 정보 요청"을 탭하면 여기에 알림이 옵니다.

각 요청 카드에는 다음 정보가 표시됩니다:
- 📋 **요청 대상 매물** 이름
- **요청자 정보**: 이름, 전화번호(📞), 이메일(✉️)
- **요청 사유** (입력한 경우)

#### 인라인 액션 버튼

| 버튼 | 기능 |
|------|------|
| ✅ **승인** | 요청 승인 → 고객에게 추가 정보 열람 권한 부여 |
| ❌ **거절** | 요청 거절 |
| 📞 **전화** | 요청자에게 바로 전화 걸기 |

승인/거절 처리 후 상태가 "✅ 승인 완료" 또는 "❌ 거절됨"으로 변경됩니다.

### 열람 기록 확인하기

고객이 내 콘텐츠를 열람하면 자동으로 기록됩니다:

| 아이콘 | 열람 유형 |
|--------|-----------|
| 📄 | 모바일 IM(투자설명서) 열람 |
| 🃏 | 딜카드(티저) 조회 |
| 📰 | 주간 매거진 열람 |
| ✨ | Vibe 명함 조회 |

각 열람 기록에는 열람자 정보가 최대한 표시됩니다:
- 로그인 열람자: 이름, 이메일
- 비로그인 열람자: IP 기반 추정 위치, 기기 종류(iPhone/Android/Mac), 유입 경로(카카오, 네이버 등)

열람 기록을 탭하면 해당 매물의 **딜카드 상세 화면**으로 바로 이동합니다.

## 10. 카카오톡 공유 기능 활용하기

### 개요
Credeal의 모든 주요 콘텐츠는 **카카오톡 1탭 공유**를 지원합니다.  
AI가 자동 생성한 공유 문구와 OG 이미지가 함께 전송되어, 고객이 받았을 때 깔끔한 프리뷰 카드로 보입니다.

### 공유 가능한 콘텐츠

| 콘텐츠 | 공유 위치 | 공유되는 URL |
|--------|-----------|-------------|
| **딜카드** | 딜카드 상세 → 하단 🟡 "카톡으로 전송" | `https://credeal.net/dc/{ID}` |
| **Vibe 명함** | 명함 관리 → 공유 → 카카오톡 | `https://credeal.net/vibe-card/{slug}` |
| **모바일 IM** | IM 생성 후 공유 버튼 | `https://credeal.net/im/{ID}` |

### 딜카드 카카오톡 공유 방법

1. 딜카드 상세 화면(`/broker/deal-card/{ID}`) 하단으로 스크롤
2. 🟡 **"카톡으로 전송"** 버튼 탭
3. 카카오톡 앱이 열리며 대화 상대 선택
4. 전송!

> 💡 전송되는 메시지에는:
> - **OG 이미지**: AI가 생성한 딜카드 프리뷰 이미지
> - **타이틀**: 매물 캐치프레이즈
> - **설명**: AI 생성 요약
> - **"딜카드 보기" 버튼**: 탭하면 블라인드 딜카드 공개 페이지 열림

### 카톡 공유 문구 수정하기

AI가 자동 생성한 카톡 문구가 마음에 들지 않는다면:
1. 딜카드 에디터에서 **💬 카톡 공유 문구** 섹션을 찾습니다
2. 텍스트를 원하는 대로 수정합니다
3. **"저장"** 버튼을 탭합니다

또는 하단 CTA 바의 카톡 버튼 옆 **✏️ 수정** 아이콘을 탭하면 바로 수정할 수 있습니다.

### 카톡 문구 복사하기

카카오톡 SDK가 작동하지 않는 환경(PC 브라우저 등)에서는:
1. **"📋 카톡 문구 복사"** 버튼을 탭
2. 클립보드에 복사됨 → **"✅ 복사됨!"** 확인
3. 카카오톡 앱에서 직접 붙여넣기

### Vibe 명함 카카오톡 공유 방법

1. `/broker/vibe-card` 에서 우측 상단 **공유 아이콘** 탭
2. **"카카오톡 공유"** 선택
3. 카카오톡이 열리면서 OG 프리뷰 카드 포함된 메시지 전송

## 자주 묻는 질문 (FAQ)

### Q. 온보딩에서 사진을 잘못 올렸어요. 다시 할 수 있나요?
**A.** 네! `/broker/vibe-card`에서 **"📷 사진 변경"** 또는 **"🔄 재분석"** 버튼을 탭하면 됩니다.

### Q. 딜카드를 만들었는데 카카오톡 전송이 안 돼요.
**A.** PC 브라우저에서는 카카오톡 SDK가 제한될 수 있습니다. **"📋 카톡 문구 복사"** 버튼으로 복사 후 카카오톡 앱에서 직접 붙여넣으세요.

### Q. 메모에서 딜카드가 자동으로 안 만들어져요.
**A.** AI가 매물 정보로 인식하려면 **지역명, 가격, 건물 유형** 등의 키워드가 포함되어야 합니다. "성수동 빌딩 80억"처럼 구체적으로 입력해 주세요.

### Q. 공유한 링크를 누가 열었는지 알 수 있나요?
**A.** 네! **🔔 소통 관리함**(`/broker/inbox`)에서 확인할 수 있습니다. 열람자의 기기 종류, 유입 경로 등이 최대한 자세히 표시됩니다.

### Q. 프로필 완성도를 빠르게 올리려면?
**A.** 자격증 번호(+15%), 거래 실적(+15%), AI 자기소개(+10%)만 입력해도 40% 이상 올라갑니다.
