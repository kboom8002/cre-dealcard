## 1. 접속하기

### 주소
브라우저에서 아래 URL로 접속합니다.

```
https://credeal.net
```

> 💡 **모바일 최적화**: Credeal은 모바일(스마트폰) 환경에 최적화된 웹 앱입니다.  
> iPhone의 Safari 또는 Android의 Chrome에서 접속하면 가장 편리합니다.

### 홈 화면에 추가하기 (선택)
모바일에서 앱처럼 빠르게 접속하려면:
- **iPhone**: Safari에서 접속 → 하단 공유 버튼(□↑) → **"홈 화면에 추가"**
- **Android**: Chrome에서 접속 → 우측 상단 메뉴(⋮) → **"홈 화면에 추가"**

홈 화면에 "JS 딜카드" 아이콘이 추가되며, 앱처럼 바로 실행할 수 있습니다.

## 2. 회원가입

### 접속 경로
```
https://credeal.net/signup
```
또는 로그인 화면 하단의 **"아직 계정이 없으신가요? → 회원가입"** 링크를 탭합니다.

### 가입 화면 안내

화면 상단에 **"회원가입"** 제목과 **"JS 딜카드 계정을 만들어 시작하세요."** 안내 문구가 표시됩니다.

### 입력 항목 (4개)

| 항목 | 설명 | 플레이스홀더 |
|------|------|-------------|
| **이름 / 닉네임** | 2~30자, 실명 또는 닉네임 (필수) | `홍길동` |
| **이메일** | 로그인 시 사용할 이메일 (필수) | `you@example.com` |
| **비밀번호** | 8자 이상 (필수) | `8자 이상` |
| **계정 유형** | 아래 두 가지 중 택 1 | — |

### 계정 유형 선택

2열 그리드 형태의 라디오 버튼 중 하나를 선택합니다:

- 👤 **일반 사용자** (기본 선택) — 매수자, 건물주 등 일반 이용자
- 🏢 **공인중개사** — 부동산 전문가 (중개사, 법무사, 세무사 등)

> ⚠️ **중요**: 중개인이시라면 반드시 **🏢 공인중개사**를 선택하세요!  
> 딜카드, 매거진, 매칭, 메모함, Vibe 명함 등 전문가 전용 기능이 활성화됩니다.

### 가입 절차

1. 모든 항목을 입력합니다
2. **"가입하기"** 버튼을 탭합니다 (버튼이 `계정 생성 중...`으로 변경)
3. 계정이 즉시 생성되며, 자동으로 **온보딩 페이지**(`/onboarding`)로 이동합니다

> 💡 화면 하단의 안내: *"회원가입 시 JS 딜카드의 이용약관 및 개인정보처리방침에 동의하게 됩니다."*

### 자주 발생하는 오류

| 오류 메시지 | 원인 | 해결 방법 |
|-------------|------|-----------|
| 이미 사용 중인 이메일입니다 | 이미 가입된 이메일 | 로그인 페이지로 이동 |
| 비밀번호는 8자 이상이어야 합니다 | 비밀번호 짧음 | 8자 이상으로 재입력 |
| 이름은 2~30자여야 합니다 | 이름 길이 부적합 | 2~30자 범위로 재입력 |

## 3. 로그인

### 접속 경로
```
https://credeal.net/login
```

### 로그인 화면 안내

화면 상단에 **"로그인"** 제목과 **"계속하려면 계정으로 로그인하세요."** 안내 문구가 표시됩니다.

### 로그인 방법

1. 가입 시 사용한 **이메일** 입력 (플레이스홀더: `you@example.com`)
2. **비밀번호** 입력 (플레이스홀더: `••••••••`)
3. **"로그인"** 버튼 탭 (버튼이 `로그인 중...`으로 변경)

로그인 성공 시:
- **온보딩 미완료**: 자동으로 `/onboarding`으로 이동
- **온보딩 완료**: 자동으로 중개인 대시보드(`/broker`)로 이동

### 비밀번호 분실 시

로그인 화면 내 **"비밀번호를 잊으셨나요?"** 링크를 탭하면 비밀번호 재설정 페이지(`/reset-password`)로 이동합니다.

> ⚠️ 이메일 또는 비밀번호가 틀리면 **"이메일 또는 비밀번호가 올바르지 않습니다."** 메시지가 표시됩니다.

### 하단 링크
**"아직 계정이 없으신가요?"** → **"회원가입"** 링크로 가입 페이지로 이동합니다.

## 4. 온보딩 제대로 하기

### 온보딩이란?
첫 로그인 후 자동으로 시작되는 **초기 설정 과정**입니다.  
AI가 프로필 사진을 분석하여 **Vibe 명함**(디지털 명함)을 자동 생성하고, 전문 분야와 활동 지역을 설정합니다.

> 📄 페이지 제목: *"내 Vibe 명함 만들기 | DealCard"*  
> 📄 설명: *"사진 한 장으로 AI가 분석하는 나만의 부동산 전문가 프로필을 만들어보세요."*

### 접속 경로
```
https://credeal.net/onboarding
```
> 이미 온보딩을 완료했다면 자동으로 대시보드로 이동합니다. 신규 가입자는 자동으로 이 페이지로 안내됩니다.

### 진행 순서 (8단계)

화면 상단의 **진행 바(●●●●●)**로 현재 진행 상황을 확인할 수 있습니다. 각 단계 사이에 부드러운 애니메이션 전환이 적용됩니다.

---

#### ① 역할 선택 (`role_select`)

두 가지 역할 카드 중 하나를 선택합니다:

| 카드 | 역할 | 설명 |
|------|------|------|
| 🏢 **부동산 전문가** (`expert`) | 공인중개사, 법무사, 세무사, 감정평가사 | 딜카드, 빌딩 레이더, 매거진 등 추가 전문가 기능 포함 |
| 🏠 **건물주 / 투자자** (`owner`) | 건물 소유주, 매수 관심자, 개발사 | 아고라(투자자 커뮤니티) 기능 포함 |

> 💡 **JS 중개인이시라면 "부동산 전문가"를 선택**하세요.

---

#### ② 프로필 사진 업로드 (`photo_upload`)

**얼굴이 보이는 프로필 사진**을 업로드합니다.

> ⚠️ **얼굴 사진 필수!** AI Vibe 분석은 사진 속 인상을 기반으로 신뢰도, 성향 프로필을 생성합니다. 풍경 사진이나 로고 등은 분석 품질이 크게 떨어집니다.

**추천 사진 조건:**
- 정면 또는 약간 비스듬한 상반신 사진
- 밝은 조명, 자연스러운 표정
- 정장 또는 깔끔한 복장
- 최소 300×300px 이상

---

#### ③ AI 분석 중 (`analyzing`)

사진 업로드 후 **약 10~20초** 동안 AI가 분석합니다.

- 화면에 업로드된 프로필 사진 미리보기가 표시됩니다
- 분석 진행 상태 바가 실시간으로 업데이트됩니다
- 내부적으로 `/api/onboarding/analyze-photo` API를 호출합니다

> 💡 분석 중에는 페이지를 닫지 마세요. 인터넷 연결이 불안정하면 분석이 실패할 수 있습니다.

---

#### ④ 분석 결과 확인 (`reveal`)

AI가 생성한 **Vibe 프로필**이 공개됩니다:

- **VTI 유형**: MBTI처럼 중개 스타일을 표현하는 아이콘과 레이블 (예: 🏛️ 자산전략가)
- **신뢰 점수 (Trust)**: 사진에서 분석된 신뢰 인상 점수
- **밸런스 점수 (Valence)**: 호감도/친근감 점수
- **Before/After**: 분석 전후 점수 비교

여기서 **"공유하기"** 버튼을 탭하면 분석 결과를 카카오톡으로 바로 공유할 수 있습니다.

공유 시 전송되는 메시지:
```
🏢 DealCard Vibe 분석
{VTI 유형} 타입
Trust {점수}

나도 해보기 → credeal.net/onboarding
```

**"계속"** 을 탭하여 다음 단계로 이동합니다.

---

#### ⑤ 상세 정보 입력 (`login` — 명함 완성)

이 단계에서는 로그인/회원가입 통합 폼이 제공됩니다. 이미 로그인된 상태라면 자동으로 다음 단계로 넘어갑니다.

**입력 항목:**
- **이름** (필수): 명함에 표시될 이름
- **연락처** (선택): 전화번호

**"나중에 하기"** 를 탭하면 건너뛸 수 있지만, 가능하면 입력을 권장합니다.

---

#### ⑥ 빌딩 레이더 (`radar`)

건물 주소를 입력하여 **AI 건물 분석**을 체험합니다.

- 주소 또는 위치 입력 후 검색
- AI가 해당 건물의 시장 시그널, 거래 현황 등을 분석

> 💡 건너뛰기를 탭해도 됩니다. 대시보드에서 언제든 이용 가능합니다.

---

#### ⑦ 경로 분기 — 역할별 체험

| 역할 | 체험 기능 | 설명 |
|------|-----------|------|
| 🏢 **부동산 전문가** | **첫 번째 딜카드** (`dealcard`) | 카카오톡에서 보내던 매물 소개 문구를 붙여넣으면 30초 만에 블라인드 딜카드 생성 (빌딩 레이더 주소가 자동 전달됨) |
| 🏠 **건물주 / 투자자** | **아고라** (`agora`) | 투자자 커뮤니티 체험 |

> 💡 두 경로 모두 건너뛰기 가능합니다.

---

#### ⑧ 온보딩 완료! 🎉 (`complete`)

축하 화면이 나타납니다! 달성한 항목을 체크리스트로 보여줍니다:
- 🎯 AI Vibe 프로필 명함
- 🏢 빌딩 레이더 첫 조회 (진행한 경우)
- 📋 블라인드 딜카드 1개 (진행한 경우)

**Vibe 명함 미리보기**가 하단에 표시됩니다. 여기서:

- **📱 카톡으로 공유하기**: 명함 링크를 카카오톡으로 바로 공유
- **대시보드로 이동**: 중개인 코크핏(대시보드)으로 이동

> 💡 **온보딩에서 입력한 이름, 연락처, 프로필 사진**은 모두 프로필에 저장됩니다. 온보딩 완료 후 대시보드에서 이름과 프로필 사진이 정상적으로 표시되는지 확인하세요.

## 5. 대시보드 (중개인 코크핏)

### 접속 경로
```
https://credeal.net/broker
```
로그인 후 자동으로 이동합니다.

### 화면 구성

#### 인사말 영역 (상단)

대시보드 최상단에 **인사말**과 **프로필 사진**이 표시됩니다.

- **이름 표시 우선순위**: 프로필 이름(`display_name`) → 브로커 프로필 이름 → 가입 시 입력한 이름 → 이메일 앞부분 → "중개인"
- 프로필 사진이 등록된 경우 함께 표시됩니다
- 우측 상단에 **프로필 아이콘**과 **🔔 알림 아이콘**이 있습니다

#### 퀵 액션 (6개 버튼, 2열 그리드)

| 아이콘 | 제목 | 부제 | 이동 경로 |
|--------|------|------|-----------|
| 📝 | **30초 딜카드** | 카톡 메모 붙여넣기 | `/broker/deal-card/new` |
| 🎯 | **매수자 등록** | 조건 메모 붙여넣기 | `/broker/buyer-intents/new` |
| ⚡ | **매칭 센터** | AI 매칭 결과 확인 | `/broker/matching` |
| 🎬 | **콘텐츠 스튜디오** | 매거진 & 카톡 공유 | `/broker/studio` |
| 📰 | **매거진 관리** | 발행 이력 · 편집 · 성과 | `/broker/magazine-editor?tab=analytics` |
| ✨ | **Vibe 명함** | 명함 관리 · 재생성 · 공유 | `/broker/vibe-card` |

#### 4개 탭 (대시보드 콘텐츠)

| 탭 | 한글명 | 주요 내용 |
|----|--------|-----------|
| **Overview** | 오버뷰 | 오늘의 일정, KPI 스크롤 카드, 알림 피드, ROI 카드, 파이프라인 성과 요약 |
| **Breakthrough** | 시장 대응 | 시장 돌파 모드 (MarketBreakthroughMode) |
| **Reports** | 리포트 | 주간 리포트 + 월간 리포트 |
| **Morning Intelligence** | 모닝 인텔 | AI 모닝 브리핑 |

#### Overview 탭 상세

**KPI 스크롤 카드** (좌우 스크롤):

| KPI | 설명 |
|-----|------|
| 매매물건 | 보유 매매 매물 수 |
| 임대물건 | 보유 임대 매물 수 |
| S/A 매칭 | S등급/A등급 자동 매칭 수 |
| 관리 고객 | 관리 중인 고객 수 |
| 매수 고객 | 등록된 매수 고객 수 |

**오늘의 일정**: `오늘의 일정 ({n}건)` — 예약 시간, 주소, 상태(`예약 확정` / `승인 대기`), 매수자 이름 표시

**최근 알림 피드**: 10종 이상의 이벤트 유형이 실시간으로 표시됩니다:
- `신규 Gate 상세 주소 요청 접수`
- `[S급 매칭] 신규 자동 매칭 발생`
- `IM 열람 기록`
- `딜카드 열람`
- `Vibe 명함 조회`
- `주간 매거진 열람` 등

**파이프라인 성과 요약** (`📈 내 파이프라인 성과 요약`):

| 지표 | 비교 대상 |
|------|-----------|
| 내 딜 평균 체류일수 | 시장 평균 대비 |
| S등급 매칭 전환율 | 35% 평균 대비 |
| 매거진 구독자 | 활성 구독자 수 |
| 이달 매거진 열람 | 페이지 뷰 기준 |

### Vibe 명함이란?
AI가 프로필 사진을 분석하여 자동으로 생성한 **디지털 명함**입니다. 7차원 성향 벡터, VTI 유형(예: 🏛️ 자산전략가), 신뢰 점수 등이 표시됩니다.

32종의 그라디언트 테마 중 AI가 성향에 맞는 템플릿을 자동 매칭하며, **Adaptive Contrast System**이 밝은/어두운 배경 모두에서 텍스트 시인성을 보장합니다.

### 관리 페이지 구성

#### 상단 헤더

- **"📇 내 Vibe 명함"** 제목
- 좌측: **"홈"** (뒤로가기) 버튼

하단 안내: *"카드를 탭하면 공개 페이지로 이동합니다"*

- **"아직 Vibe 분석이 없습니다"** 메시지 표시

### Adaptive Contrast System

32종의 템플릿에는 밝은 배경(워밍 코랄 등)과 어두운 배경(미드나잇 블루 등)이 혼재합니다. **Adaptive Contrast System**이 배경 luminance를 실시간 계산하여:

- **밝은 배경**: 어두운 텍스트 + 밝은 glassmorphism overlay
- **어두운 배경**: 밝은 텍스트 + 어두운 glassmorphism overlay

를 자동 적용하므로, 어떤 템플릿에서도 이름·타이틀·연락처가 선명하게 보입니다.

## 9. 딜카드 기능 활용하기

### 딜카드란?
카카오톡에서 보내던 매물 소개 문구를 붙여넣으면, AI가 자동으로 **민감정보를 숨긴 블라인드 딜카드**를 생성합니다.

### 딜카드 만들기

#### 접속 경로
```
https://credeal.net/broker/deal-card/new
```
또는 대시보드의 **"📝 30초 딜카드"** 퀵 액션 버튼을 탭합니다.

#### 생성 과정

**Step 1. 매물 정보 붙여넣기**
- 큰 텍스트 입력창이 나타납니다
- **평소 카카오톡으로 보내던 매물 소개 문구 그대로** 붙여넣으세요
- 예시 메모 보기: **"예시 메모 사용"** 버튼을 탭하면 샘플 텍스트가 자동 입력됩니다
- 메모함에서 **"딜카드 만들기 →"**로 전환한 경우, 메모 텍스트가 자동 입력됩니다

> 💡 입력 예시:
> ```
> 성수동 카페거리 5층 꼬마빌딩
> 매매가 80억 (감정가 75억)
> 1층 스타벅스, 2~4층 사무실, 5층 공실
> 보증금 총 8억 / 월세 3,200만
> 연 수익률 약 4.8%
> 건물주 해외 이주로 급매
> ```

**Step 2. 사진 첨부 (선택)**
- 건물 사진을 첨부할 수 있습니다 (여러 장 가능)
- 딜카드에 사진이 갤러리로 표시됩니다

**Step 3. "1분 딜카드 만들기" 버튼 탭**

AI가 5단계로 처리합니다 (약 30~60초):
1. ✅ 메모에서 매물 정보 추출 중
2. ✅ 건물 기본 신호 생성 중
3. ✅ 민감정보 식별 및 숨김 처리 중
4. ✅ 블라인드 딜카드 생성 중
5. ✅ 카카오 문구 준비 중

> ⚠️ 입력 내용에 **지역, 자산 유형, 숫자(가격/수익률), 거래 유형** 중 하나라도 빠지면 품질 경고가 표시됩니다. 가능하면 모두 포함시켜 주세요.

**Step 4. 생성 완료!**
세 가지 다음 행동이 제안됩니다:
1. 🟡 **딜카드 확인 및 카톡 공유** → 딜카드 상세 페이지로 이동
2. 📅 **임장 스케줄 설정하기** → 스케줄 관리
3. 📋 **내 딜카드 목록 보기** → 전체 매물 목록

```
https://credeal.net/broker/deal-card/{딜카드ID}
```

## 11. 카카오톡 공유 기능 활용하기

### 개요
Credeal의 모든 주요 콘텐츠는 **카카오톡 1탭 공유**를 지원합니다.  
카카오 SDK를 통한 **리치 메시지**(OG 이미지 + 제목 + 설명 + 버튼)가 전송되며, SDK가 작동하지 않는 환경에서는 Web Share API → 클립보드 복사 순서로 자동 fallback됩니다.

### 공유 가능한 콘텐츠

| 콘텐츠 | 공유 위치 | 공유되는 URL | OG 제목 포맷 |
|--------|-----------|-------------|-------------|
| **딜카드** | 딜카드 상세 → 하단 🟡 "카톡으로 전송" | `https://credeal.net/dc/{ID}` | 매물 캐치프레이즈 |
| **Vibe 명함** | 명함 관리 → 공유 → 카카오톡 | `https://credeal.net/vibe-card/{slug}` | `이름 — 타이틀` |
| **모바일 IM** | IM 생성 후 공유 버튼 | `https://credeal.net/im/{ID}` | 건물명 + 투자 요약 |

### 카카오 SDK vs OG 메타태그

Credeal은 두 가지 방식으로 카카오톡 공유 정보를 제공합니다:

| 방식 | 적용 시점 | 특징 |
|------|----------|------|
| **카카오 SDK** (`sendDefault`) | 앱 내 공유 버튼 탭 시 | 리치 메시지 (이미지, 제목, 설명, 버튼) 직접 전달 |
| **OG 메타태그** | URL을 카톡에 직접 붙여넣을 때 | 카카오 크롤러가 페이지에서 og:title, og:image 등을 파싱 |

> 💡 **둘 다 동일한 정보**가 표시되도록 코드가 동기화되어 있습니다. 이름/타이틀을 변경하면 SDK 공유와 URL 붙여넣기 모두 새 정보가 반영됩니다.

### Vibe 명함 카카오톡 공유 상세

#### 공유 방법
1. `/broker/vibe-card` 에서 우측 상단 **공유 아이콘** 탭
2. **"카카오톡 공유"** 선택 (SDK 연결 시) 또는 **"카톡 문구 복사"** (SDK 미연결 시)
3. 카카오톡이 열리면서 리치 메시지 전송

#### 카카오톡에 표시되는 정보

```
┌──────────────────────────────┐
│  [OG 이미지: 1200×630]       │
│  Vibe 명함 프리뷰 + 프로필    │
├──────────────────────────────┤
│  이상은 — CRE 투자 전문가     │  ← 제목
│  강남·서초 전문 | 경력 12년    │  ← 설명
│  ─────────────────────────── │
│  [  명함 보기  ]              │  ← 버튼
│  cre-dealcard                │
└──────────────────────────────┘
```

- **제목**: `{이름} — {타이틀}` (커스텀 타이틀 반영)
- **설명**: 전문 지역 · 전문 자산 유형 · 경력 · 소속 (점수/지수 등 대외적으로 어색한 수치 미포함)
- **OG 이미지**: Vibe 템플릿 배경 + 프로필 사진 + 이름 + 전문 분야 (Adaptive Contrast 적용)
- **버튼**: "명함 보기" → 공개 명함 페이지 열림

#### OG 캐시 초기화

이름이나 타이틀을 변경한 후 카카오톡에 즉시 반영하려면:
1. [카카오 OG 디버거](https://developers.kakao.com/tool/debugger/sharing) 접속
2. 명함 URL 입력 (예: `https://credeal.net/vibe-card/홍길동-abc123`)
3. **"캐시 초기화"** 클릭
4. 이후 공유하면 새 정보가 반영됩니다

> ⚠️ 카카오 OG 디버거를 사용하지 않으면, 기존 캐시가 만료될 때까지 (수 시간~1일) 이전 정보가 보일 수 있습니다.

### 딜카드 카카오톡 공유 방법

1. 딜카드 상세 화면(`/broker/deal-card/{ID}`) 하단으로 스크롤
2. 🟡 **"카톡으로 전송"** 버튼 탭
3. 카카오톡 앱이 열리며 대화 상대 선택
4. 전송!

> 💡 전송되는 메시지에는:
> - **OG 이미지**: AI가 생성한 딜카드 프리뷰 이미지
> - **타이틀**: 매물 캐치프레이즈
> - **설명**: AI 생성 요약
> - **"딜카드 보기" 버튼**: 탭하면 블라인드 딜카드 공개 페이지 열림

### 카톡 공유 문구 수정하기

AI가 자동 생성한 카톡 문구가 마음에 들지 않는다면:
1. 딜카드 에디터에서 **💬 카톡 공유 문구** 섹션을 찾습니다
2. 텍스트를 원하는 대로 수정합니다
3. **"저장"** 버튼을 탭합니다

또는 하단 CTA 바의 카톡 버튼 옆 **✏️ 수정** 아이콘을 탭하면 바로 수정할 수 있습니다.

### 카톡 문구 복사하기 (SDK 미연결 시)

카카오톡 SDK가 작동하지 않는 환경(PC 브라우저 등)에서는:
1. **"📋 카톡 문구 복사"** 버튼을 탭
2. 클립보드에 복사됨 → **"✅ 복사됨!"** 확인
3. 카카오톡 앱에서 직접 붙여넣기

복사되는 문구 예시:
```
[DealCard 명함]

이상은 — CRE 투자 전문가

🔗 https://credeal.net/vibe-card/이상은-abc123

📊 강남·서초 전문 | 경력 12년 — DealCard Vibe AI 명함
```

## 자주 묻는 질문 (FAQ)

### Q. 온보딩에서 사진을 잘못 올렸어요. 다시 할 수 있나요?
**A.** 네! `/broker/vibe-card`에서 **"📷 사진 변경"** 또는 **"🔄 재분석"** 버튼을 탭하면 됩니다.

### Q. 명함에 "공인중개사" 대신 다른 타이틀을 표시하고 싶어요.
**A.** `/broker/vibe-card`에서 카드 프리뷰 아래의 **"명함 표시 정보"** 섹션에서 타이틀을 자유롭게 변경할 수 있습니다. 예: "CRE 투자 전문가", "상업용 부동산 컨설턴트", "법무사" 등.

### Q. 명함에 닉네임 대신 실명을 표시하고 싶어요.
**A.** 같은 **"명함 표시 정보"** 섹션에서 이름을 변경하세요. 프로필 닉네임과 별도로 관리됩니다.

### Q. 카카오톡으로 공유했는데 이전 이름/타이틀이 보여요.
**A.** 카카오톡이 OG 정보를 캐시하기 때문입니다. [카카오 OG 디버거](https://developers.kakao.com/tool/debugger/sharing)에서 해당 URL의 **캐시를 초기화**한 후 다시 공유하세요.

### Q. 딜카드를 만들었는데 카카오톡 전송이 안 돼요.
**A.** PC 브라우저에서는 카카오톡 SDK가 제한될 수 있습니다. **"📋 카톡 문구 복사"** 버튼으로 복사 후 카카오톡 앱에서 직접 붙여넣으세요.

### Q. 메모에서 딜카드가 자동으로 안 만들어져요.
**A.** AI가 매물 정보로 인식하려면 **지역명, 가격, 건물 유형** 등의 키워드가 포함되어야 합니다. "성수동 빌딩 80억"처럼 구체적으로 입력해 주세요.

### Q. 공유한 링크를 누가 열었는지 알 수 있나요?
**A.** 네! **🔔 소통 관리함**(`/broker/inbox`)에서 확인할 수 있습니다. 열람자의 기기 종류, 유입 경로 등이 최대한 자세히 표시됩니다.

### Q. 프로필 완성도를 빠르게 올리려면?
**A.** 자격증 번호(+15%), 거래 실적(+15%), AI 자기소개(+10%)만 입력해도 40% 이상 올라갑니다.

### Q. 로고 파일이 JPEG인데 업로드 가능한가요?
**A.** Vibe 명함 로고에는 PNG, WebP, SVG만 지원됩니다. JPEG 파일은 로고 업로드에 사용할 수 없으며, 투명 배경 PNG가 가장 이상적입니다.

### Q. 밝은 배경 템플릿에서 글씨가 안 보여요.
**A.** Adaptive Contrast System이 자동으로 배경 밝기에 따라 텍스트 색상과 그림자를 조정합니다. 문제가 계속되면 프로필 사진을 재분석(🔄)하여 다른 템플릿이 매칭되게 할 수 있습니다.

### Q. 메모함에서 과거 메모를 찾고 싶어요.
**A.** 메모함 상단의 **검색 입력창**에 키워드를 입력하면 실시간으로 메모를 검색할 수 있습니다. 또한 분류 탭(매물/매수/일반)을 활용하면 유형별로 빠르게 찾을 수 있습니다. 중요한 메모는 **📌 고정** 기능을 활용하세요.

### Q. 메모를 수정할 수 있나요?
**A.** 네! 메모 카드의 **⋮ 메뉴 → "수정하기"**를 탭하면 인라인 편집 모드에서 바로 수정할 수 있습니다. 수정 이력은 타임스탬프에 `수정됨` 표시로 기록됩니다.

### Q. Vibe 명함 FAQ는 어떻게 관리하나요?
**A.** `/broker/vibe-card` 페이지의 **"자주 묻는 질문 (FAQ)"** 섹션에서 1~7개의 질문/답변을 직접 입력하고 저장할 수 있습니다. 입력하지 않으면 공개 명함에 FAQ 섹션이 표시되지 않습니다.
