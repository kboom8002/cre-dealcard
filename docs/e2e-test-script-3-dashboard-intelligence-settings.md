# 🧪 E2E 데모 테스트 스크립트 3/3 — 대시보드·인텔리전스·프로필·공개 페이지 (Dashboard, Intelligence & Settings)

> **범위**: 코크핏 대시보드(4탭) → 모닝 인텔리전스 → 주간/월간 리포트 → ROI → 프로필 관리 → 공개 페이지 → 시장 인텔리전스  
> **대상**: 브로커 사용자 (로그인 상태) + 비인증 외부 사용자 (공개 페이지)  
> **환경**: Production (`https://credeal.net`)  
> **전제 조건**: 스크립트 1~2에서 딜카드, 의향서, CRM 데이터가 일정량 생성되어 있을 것
