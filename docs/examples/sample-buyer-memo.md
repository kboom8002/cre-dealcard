# Sample Buyer Memo Lite

## Input Context

Buyer Intent:

```text
50억~80억, 성수/강남 선호, 사옥 사용 + 일부 임대수익, 주차 필수, 임차인 만기 중요.
```

Building Signal:

```text
성수권역 80억대 근생형 자산. 사옥+부분임대형 가설 가능. 일부 임대 중. 주차와 임대차 확인 필요.
```

---

## Buyer Memo

대표님, 말씀 주신 사옥+일부 임대수익 조건 기준으로 보면 이 후보는 일부 검토 가치가 있습니다.

### 맞는 점

- 예산 범위와 대략적으로 부합할 수 있습니다.
- 성수권역 선호 조건과 맞습니다.
- 사옥+부분임대 시나리오를 검토해볼 수 있습니다.

### 주의할 점

- 주차 조건은 반드시 확인이 필요합니다.
- 기존 임차인 만기와 실사용 가능 시점 확인이 필요합니다.
- 건물 노후도와 리모델링 비용은 추가 검토가 필요합니다.

### 확인 필요한 자료

- 임대차 요약표
- 건물 사진 및 도면
- 주차 현황
- 건축물대장 및 위반건축물 여부

### 다음 액션

Listing 측에 Gate G3 수준의 Snapshot 또는 IM Lite 자료를 요청하는 것이 좋습니다.

### 카톡 문구

대표님, 말씀 주신 사옥+일부 임대수익 조건 기준으로 성수권역 80억대 근생형 자산 하나가 일부 검토 가능해 보입니다. 다만 주차 조건, 임차인 만기, 실사용 가능 면적은 확인이 필요합니다. 원하시면 우선 블라인드 요약자료 기준으로 더 검토해보겠습니다.

### Boundary Note

본 메모는 예비 검토용이며, 매수 추천·가격 적정성·대출 가능성·법률/세무 판단을 확정하지 않습니다.
