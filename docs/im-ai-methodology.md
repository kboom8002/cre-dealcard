## 2. 모바일 IM — 7섹션 방법론

### 데이터 흐름

```
중개인 메모 → [DealCard AI] → building_ssot_lite
                                    ↓
                        [MobileIM Writer] → 7섹션 마크다운
                                    ↓
                        [Lite Gate] → 공시·리스크 검증
                                    ↓
                        카카오 공유 링크 생성
```

### 섹션별 데이터 소스 + AI 역할

| # | 섹션 | 소스 | AI 역할 |
|---|---|---|---|
| 1 | **물건 개요** | `asset_identity` + 사진 | SSoT에서 자동 구성 |
| 2 | **입지·상권** | `market_location.location_analysis` | 🔶 AI: 위치 분석 자동 생성 |
| 3 | **임대 현황** | `supplemental.monthly_rent` + `vacancy_signal` | SSoT + 보충 입력 |
| 4 | **수익 분석** | (추정) | ⚠️ 면책 자동 첨부 |
| 5 | **확인 필요** | (자동) | 🔶 AI: 리스크 자동 도출 |
| 6 | **투자 포인트** | `buyer_fit.fit_summary` | 🔷 **AI: 매수자 적합성 분석** |
| 7 | **다음 단계** | (정적) | Full IM 업그레이드 CTA |

### 🔷 AI 차별화 포인트

**섹션 6 투자 포인트** — `buyer_fit.fit_summary`는 dealcard의 `DealCuriosityWriter`가 생성합니다. 이 AI 에이전트는 매물의 숨겨진 가치(숨은 정보)를 분석하여 "왜 이 건물을 사야 하는지"를 자동으로 작성합니다. **경쟁사에서는 중개인이 직접 작성해야 하는 부분**입니다.

## 4. 🔷 AI 현저 차별화 — 10대 핵심 기능

### ① 메모 → SSoT 자동 변환 (DealCard)

| 항목 | 경쟁사 | 이 시스템 |
|---|---|---|
| 입력 | 엑셀 30분 입력 | **메모 1줄 2분** |
| AI | 없음 | `BuildingMiniTruthAgent` 8개 레이어 추출 |
| 정확도 | 사람 의존 | AI 추론 + 중개인 확인 |

**IM 영향**: 모바일 IM이 2분 만에 완성됨 → 매물 등록 당일 카카오 공유 가능

---

### ② 매수자 적합성 자동 매칭 (DealCard P-D2 + Matching)

| 항목 | 경쟁사 | 이 시스템 |
|---|---|---|
| 방법 | 중개인 감각 | 3-Stage 매칭 (규칙→시맨틱→가중치) |
| 결과 | "맞을 것 같다" | **S/A/B/C 등급 + 이유 + 점수** |
| IM 반영 | 없음 | 섹션3 투자논거에 매수자 유형별 맞춤 |

**IM 영향**: 섹션3 "투자 논거 및 매수자 적합성"이 **데이터 기반으로 자동 생성**됨. P-D2 클러스터(법인사옥/수익투자/증여 등)에 따라 강조점이 달라짐.

---

### ③ 딜 호기심 점수 + 숨은 정보 (DealCard)

| 항목 | 경쟁사 | 이 시스템 |
|---|---|---|
| 투자 포인트 | 중개인이 수동 작성 | `DealCuriosityWriter` AI 자동 |
| 숨은 가치 | 발견 못함 | AI가 9개 숨은 정보 카테고리 분석 |

**IM 영향**: 모바일 IM 섹션6, Full IM 섹션2·3에 "왜 이 건물인가" 자동 서술

---

### ④ 공간 AI 실증 데이터 (AiPage → IM)

| 항목 | 경쟁사 | 이 시스템 |
|---|---|---|
| 임대 수요 | "수요 많음" (주관) | **AI 검증 수요 데이터** |
| 임차인 적합 | 없음 | `TenantFitAgent` 점수 + 이유 |
| 공간 감성 | 없음 | `VibeFitAgent` VAD 프로파일 |
| 시각 증거 | 사진 나열 | `VisualAlbumAgent` 자동 앨범 |

**IM 영향**: 
- 섹션7 "미시 시장·수요"에 **실제 임대 문의 데이터 + AI 적합도 점수** 삽입
- 섹션8 "건물 상태"에 **AI 분류된 사진 앨범** 자동 첨부
- 이 데이터는 `vacancy-signal-enricher.ts`를 통해 dealcard에도 연결

### ⑤ 실거래 기반 가격 검증 (P-D)

| 항목 | 경쟁사 | 이 시스템 |
|---|---|---|

**IM 영향**: 섹션12 "가치평가·비교사례"에 **데이터 기반 가격 구간** 자동 제시 → 전문가가 패치할 때 참고 기준이 됨
