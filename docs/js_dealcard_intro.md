# 📱 JS 딜카드 — 모바일 AI 중개 자동화 플랫폼

## 개요

**카톡 한 줄이면 끝. AI가 딜카드 만들고, 매수자 찾고, 투자설명서까지.**

제이에스부동산중개 중개인이 매일 카카오톡으로 주고받는 매물 메모를,
1분 만에 블라인드 딜카드 → 매수자 매칭 → 투자설명서(IM)까지 자동 생성하는
**모바일-퍼스트 AI 업무 자동화 시스템**입니다.

---

## 7줄 핵심 설명

### ① 카톡 메모 → 1분 딜카드
매물 메모를 붙여넣으면 AI가 민감정보는 자동 마스킹하고,
블라인드 딜카드를 즉시 생성합니다.

### ② 매수자 AI 자동 매칭
등록된 매수자 조건과 매물을 S·A·B·C 등급으로 자동 매칭해,
가장 유력한 매수 후보를 먼저 보여줍니다.

### ③ 모바일 투자설명서 (IM Lite)
증빙 자료를 올리면 AI가 10개 섹션 투자설명서 초안을
약 3분 만에 생성합니다. 별도 외주가 필요 없습니다.

### ④ 거래 진행 현황판
메모입력 → 딜카드 → 열람승인 → IM 작성 → 미팅 → LOI → 계약,
모든 거래 진행 상황을 한 화면에서 추적합니다.

### ⑤ 3단계 열람 승인
매수자 자격에 따라 기본정보 → 임대상세 → 투자자료 순으로
단계별로 정보를 공개해, 매도인 정보를 안전하게 보호합니다.

### ⑥ 아침 브리핑 대시보드
출근 후 1분, 오늘의 매물·매칭·거래 진행 현황과
시장 동향을 한눈에 확인합니다.

### ⑦ 사내 공동중개 활성화
우리 회사 중개인끼리 매물·매수자를 안전하게 공유해,
외부 유출 없이 사내 공동중개를 활성화합니다.

---

> **한 마디로:** 중개인의 머릿속과 카톡에만 있던 매물 정보를,
> AI가 **구조화 → 보호 → 매칭 → 문서화**하는 올인원 모바일 도구입니다.
