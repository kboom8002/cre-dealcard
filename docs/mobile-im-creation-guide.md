## 1. 모바일 IM이란?

**모바일 IM(Investment Memorandum)**은 기존 30~50페이지의 투자설명서를 **7섹션 모바일 최적화 포맷**으로 자동 생성하는 크리딜의 핵심 기능입니다.

### 핵심 가치
- ⏱️ **작성 시간**: 기존 수일 → **5분 이내** 자동 생성
- 📱 **모바일 퍼스트**: 카카오톡/LINE으로 즉시 공유 가능
- 🔒 **정보 보호**: 민감 정보 자동 블라인드 처리
- 📊 **데이터 기반**: 건축물대장, 토지대장, 공시지가 등 공공데이터 자동 연동
- 🤖 **AI 분석**: GPT 기반 투자 분석 + 리스크 검토 + DCF 수익 분석
- 🗺️ **정확한 지도**: 카카오 로컬 API 기반 한국 주소 지오코딩

### 어떤 물건에 적합한가?
| 유형 | 적합도 | 설명 |
|------|--------|------|
| 꼬마빌딩 (30~200억) | ⭐⭐⭐⭐⭐ | 가장 최적화된 대상 |
| 상가 (10~50억) | ⭐⭐⭐⭐⭐ | 임대차 분석 강점 |
| 물류센터 | ⭐⭐⭐⭐ | 전용 물류 필드 지원 |
| 오피스빌딩 | ⭐⭐⭐⭐ | NOI/Cap Rate 분석 강점 |
| 토지/개발부지 | ⭐⭐⭐ | 기본 분석 가능 |

## 2. 작성 전 준비사항

### 필수 준비물
| 항목 | 설명 | 데이터 충실도 점수 영향 |
|------|------|----------------------|
| ✅ 매각 희망가 | 만원 단위 입력 | +15점 |
| ✅ 공실 현황 | 만실/~10%/~20% 등 선택 | +10점 |
| ✅ 건물 대표 사진 1장 이상 | 외관, 내부, 입지 등 | +10점 |
| ✅ 중개인 한줄 코멘트 | 물건의 핵심 매력 포인트 | +5점 |

### 선택 준비물 (높은 충실도 달성 시)
| 항목 | 설명 | 점수 영향 |
|------|------|---------| 
| 선순위 대출 잔액 | 만원 단위 | +10점 |
| 층별 임대차 현황 | 층/업종/보증금/월세/계약기간 | +15점 |
| 보증금 합계 | 만원 단위 | +5점 |
| 관리비 합계 | 만원 단위 | +5점 |

> **💡 팁**: 데이터 충실도 60점 이상이면 IM 자동 생성 가능. 80점 이상이면 고품질 IM 생성!

## 3. 모바일 IM 생성 Step-by-Step

### Step 1: 딜카드 상세 페이지 접근
```
메모함 → 딜카드 선택 → 상세 페이지
```

### Step 2: "🚀 모바일 투자설명서 만들기" 버튼 클릭
- 하단의 초록색 버튼을 탭합니다
- "투자설명서 데이터 보강" 바텀시트가 열립니다

### Step 3: 핵심 데이터 입력

#### 3-1. 매각 희망가 & 대출 잔액
- **매각 희망가**: 만원 단위로 입력 (예: 550000 = 55억)
- **선순위 대출 잔액**: 만원 단위 (예: 350000 = 35억)

#### 3-2. 공실 현황
4가지 중 선택: 만실 / ~10% / ~20% / 직접 입력

#### 3-3. 건물 대표 사진
- 최대 **12장** 업로드 가능
- 각 사진에 캡션은 승인 편집 페이지에서 추가 가능

#### 3-4. 중개인 한줄 코멘트
- 예: "상업지역, 리모델링 여지가 큽니다"

### Step 4: 데이터 충실도 확인
- ✅ "투자설명서 작성 가능" (60점 이상)
- ⚠️ "데이터 보강 필요" (60점 미만)

### Step 5: IM 생성 시작
- AI 분석 ~30~60초 소요

### Step 6: IM 승인 및 발행
- 생성 완료 후 **승인 페이지**로 자동 이동

## 4. 7섹션 구성과 각 섹션 설명

### 섹션 1: 🏢 이 건물, 어떤 자산인가?
| 포함 정보 | 데이터 출처 |
|-----------|-----------|
| 자산 유형, 권역, 규모 | 브로커 메모 + AI |
| 건축물대장 요약 | 국토부 API |
| 대지면적, 연면적, 용적률, 건폐율 | 건축물대장 API |

### 섹션 2: 📍 입지와 접근성은?
| 포함 정보 | 데이터 출처 |
|-----------|-----------|
| 최근접 지하철역/거리 | 카카오 POI API |
| 용도지역/용도지구 | 토지이음 API |
| 공시지가 | 공시지가 API |
| 위치 지도 | **카카오 로컬 API 지오코딩** + 카카오맵 |

### 섹션 3: 💰 현재 임대차 현황은?
| 포함 정보 | 데이터 출처 |
|-----------|-----------|
| 층별 임대차 현황 (Rent Roll) | 브로커 직접 입력 |
| WALE 자동 계산 | 자동 계산 |

### 섹션 4: 📊 수익 구조 분석
NOI, Cap Rate, 레버리지 수익률, 10년 DCF 분석, WACC, 자기자본 소요액

### 섹션 5: ⚠️ 숨은 리스크는 없는가?
리스크 요인 분석, 실사 체크리스트, 교차검증

### 섹션 6: 🎯 왜 지금 이 매물을 사야 하는가?
투자 가설, Value-Add 시나리오

### 섹션 7: 🔧 검토 후 다음 단계는?
다음 절차 안내, 중개인 연락처, 프라이빗 IM 신청 CTA

## 7. IM 승인 및 편집 (고도화)

IM 생성 후 자동으로 **승인 페이지**(`/broker/im-approval/[id]`)로 이동합니다.

### 7-1. 📊 데이터 품질 모니터링 패널 (v3.0 NEW)

승인 페이지 상단에 **편집자 전용** 모니터링 패널:
- **Readiness Score**: 바텀시트 입력 데이터 기반 IM 생성 준비도
- **Data Quality**: A~D 등급 표시
  - 🟢 **A등급** — 투자 검토 가능 (매각가 + 임대료 + 공공데이터)
  - 🟡 **B등급** — 기본 수익률 산출
  - 🟠 **C등급** — 건물 정보만
  - 🔴 **D등급** — 데이터 보충 필요

> 이 패널은 **편집자만** 볼 수 있으며, 공개 IM에는 표시되지 않습니다.

### 7-2. 📷 사진 캡션 편집 (v3.0 NEW)

- 사진이 그리드 형태로 표시 (3~4열)
- 각 사진 아래 캡션 입력란
- 캡션 입력 후 포커스 해제 시 자동 저장
- 공개 IM 갤러리 하단에 자동 표시

### 7-3. ✏️ 모바일 마크다운 에디터 (v3.0 개선)

- **PC**: 좌/우 Split-pane (편집기 + 미리보기)
- **모바일**: `✏️ 편집` / `👁 미리보기` **탭 전환** 방식
- 편집기 높이 자동 조절 (최소 300px)

### 7-4. 👁 섹션 숨김 기능 (v3.0 NEW)

각 섹션 헤더에 **👁 토글 버튼**:

| 상태 | 버튼 | 동작 |
|------|------|------|
| 공개 | `👁` | 클릭 시 숨김 |
| 숨김 | `👁‍🗨 숨김` (빨간색) | 클릭 시 해제 |

- 숨긴 섹션은 삭제되지 않음 — 편집 화면에서 반투명 표시
- 공개 IM 뷰어에서는 완전히 제외

### 7-5. 기존 기능

| 기능 | 설명 |
|------|------|
| 섹션별 검토 | 신뢰도 배지 확인 |
| 인라인 편집 | 마크다운 에디터 + 미리보기 |
| 섹션별 확인 | "미확인" ↔ "✓ 확인완료" 토글 |
| 제목 편집 | 상단 제목 클릭하여 수정 |
| 자동 저장 | 수정 시 자동 저장 |

### 발행 조건
- **모든 섹션이 "✓ 확인완료"** 상태여야 "🚀 승인 및 공개" 가능

## 8. IM 뷰어 기능 총정리

### 모바일 뷰어 UI
- **아코디언 네비게이션**: 7섹션을 펼치기/접기로 탐색
- **맵 통합**: 카카오 로컬 API 기반 정확한 위치 + 카카오맵 연동
- **사진 갤러리**: 대표 사진 슬라이더 + 라이트박스 + **캡션 표시**
- **숨김 섹션 필터**: 브로커가 숨긴 섹션은 뷰어에 미노출

> **📌 v3.0 변경**: 공개 뷰어에서 데이터 등급 배지, SSoT 완성도 바, 한국어/영어 전환 탭이 제거되었습니다.

### 공유 버튼 (하단 고정)
| 버튼 | 기능 |
|------|------|
| 🟡 카카오톡 | 카카오톡으로 IM 링크 공유 |
| 🟢 LINE | LINE으로 공유 |
| 📋 링크복사 | IM URL 클립보드 복사 |
| 📄 PDF 저장 | PDF 변환 후 다운로드 |

### 담당 중개인 카드 (Vibe 명함)
- IM 하단에 중개인의 **Vibe 명함** 자동 표시
- 프로필 사진, 전문 분야, 이메일, 전화번호

## 12. AI 엔진 작동 원리

### AI 파이프라인 흐름
```
브로커 메모 → MemoParser → BuildingMiniTruth → 외부 데이터 연동
→ 충실도 계산 → 7섹션 AI 내러티브 → 가드레일 3중 방어
→ 교차 검증 → 품질 게이트 → 발행
```

### 가드레일 3중 방어
| 가드레일 | 역할 |
|---------|------|
| **Risk Boundary** | 확정적 투자 수익 약속 방지 |
| **Disclosure Guard** | 민감 정보 자동 블라인드 |
| **Hallucination Guard** | 비정상 수치 탐지 |

## 14. 자주 묻는 질문

### Q: IM 생성에 얼마나 걸리나요?
A: **30초~60초** 소요. Vercel Pro에서 최대 5분까지 처리됩니다.

### Q: 생성된 IM을 수정할 수 있나요?
A: 승인 페이지에서 인라인 편집 가능. PC는 Split-pane, 모바일은 편집/미리보기 탭 전환.

### Q: 특정 섹션을 숨길 수 있나요?
A: 네. 승인 페이지에서 각 섹션의 **👁 토글**을 누르면 공개 IM에서 숨겨집니다.

### Q: 사진에 캡션을 달 수 있나요?
A: 네. 승인 편집 페이지에서 각 사진 아래 캡션 입력란에 텍스트를 입력하면 됩니다.

### Q: IM을 삭제할 수 있나요?
A: 네. IM 보관함에서 🗑 버튼으로 삭제. 영구적이며 복구 불가.

### Q: 지도에 정확한 위치가 표시되나요?
A: 네. **카카오 로컬 API**를 사용하여 한국 주소를 정확하게 지오코딩합니다.
