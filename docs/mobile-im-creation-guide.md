## 1. 모바일 IM이란?

**모바일 IM(Investment Memorandum)**은 기존 30~50페이지의 투자설명서를 **7섹션 모바일 최적화 포맷**으로 자동 생성하는 크리딜의 핵심 기능입니다.

### 핵심 가치
- ⏱️ **작성 시간**: 기존 수일 → **5분 이내** 자동 생성
- 📱 **모바일 퍼스트**: 카카오톡/LINE으로 즉시 공유 가능
- 🔒 **정보 보호**: 민감 정보 자동 블라인드 처리
- 📊 **데이터 기반**: 건축물대장, 토지대장, 공시지가 등 공공데이터 자동 연동
- 🤖 **AI 분석**: GPT 기반 투자 분석 + 리스크 검토 + DCF 수익 분석
- 🗺️ **정확한 지도**: 카카오 로컬 API 기반 한국 주소 지오코딩
- 🖼️ **OG 이미지 자동 생성**: 1200×630 정규 OG 이미지로 카카오톡 공유 최적화 (v4.0)

### 어떤 물건에 적합한가?
| 유형 | 적합도 | 설명 |
|------|--------|------|
| 꼬마빌딩 (30~200억) | ⭐⭐⭐⭐⭐ | 가장 최적화된 대상 |
| 상가 (10~50억) | ⭐⭐⭐⭐⭐ | 임대차 분석 강점 |
| 물류센터 | ⭐⭐⭐⭐ | 전용 물류 필드 지원 |
| 오피스빌딩 | ⭐⭐⭐⭐ | NOI/Cap Rate 분석 강점 |
| 토지/개발부지 | ⭐⭐⭐ | 기본 분석 가능 |

## 2. 작성 전 준비사항

### 필수 준비물
| 항목 | 설명 | 데이터 충실도 점수 영향 |
|------|------|----------------------|
| ✅ 매각 희망가 | 만원 단위 입력 | +15점 |
| ✅ 공실 현황 | 만실/~10%/~20% 등 선택 | +10점 |
| ✅ 건물 대표 사진 1장 이상 | 외관, 내부, 입지 등 | +10점 |
| ✅ 중개인 한줄 코멘트 | 물건의 핵심 매력 포인트 | +5점 |

### 선택 준비물 (높은 충실도 달성 시)
| 항목 | 설명 | 점수 영향 |
|------|------|---------|
| 선순위 대출 잔액 | 만원 단위 | +10점 |
| 층별 임대차 현황 | 층/업종/보증금/월세/계약기간 | +15점 |
| 보증금 합계 | 만원 단위 | +5점 |
| 관리비 합계 | 만원 단위 | +5점 |

> **💡 팁**: 데이터 충실도 60점 이상이면 IM 자동 생성 가능. 80점 이상이면 고품질 IM 생성!

## 3. 모바일 IM 생성 Step-by-Step

### Step 1: 딜카드 상세 페이지 접근
```
메모함 → 딜카드 선택 → 상세 페이지
```

### Step 2: "🚀 모바일 투자설명서 만들기" 버튼 클릭
- 하단의 초록색 버튼을 탭합니다
- "투자설명서 데이터 보강" 바텀시트가 열립니다

### Step 3: 핵심 데이터 입력

#### 3-1. 매각 희망가 & 대출 잔액
- **매각 희망가**: 만원 단위로 입력 (예: 550000 = 55억)
- **선순위 대출 잔액**: 만원 단위 (예: 350000 = 35억)

#### 3-2. 공실 현황
4가지 중 선택: 만실 / ~10% / ~20% / 직접 입력

#### 3-3. 건물 대표 사진
- 최대 **12장** 업로드 가능
- 각 사진에 캡션은 승인 편집 페이지에서 추가 가능

#### 3-4. 중개인 한줄 코멘트
- 예: "상업지역, 리모델링 여지가 큽니다"

### Step 4: 데이터 충실도 확인
- ✅ "투자설명서 작성 가능" (60점 이상)
- ⚠️ "데이터 보강 필요" (60점 미만)

### Step 5: IM 생성 시작
- AI 분석 ~30~60초 소요
- AI가 자동으로 **CRE IM 업계 표준 문체**로 제목을 생성합니다 (v4.0)
  - 변경 전: `"중구권역 근린생활시설 또는 다가구·상가주택 계열로 추정 투자설명서"`
  - 변경 후: `"중구권역 근린생활시설 매각"`
  - `"~계열로 추정"`, `"또는 ~"` 등 분석적/기계적 어투가 자동으로 제거됩니다

### Step 6: IM 승인 및 발행
- 생성 완료 후 **승인 페이지**로 자동 이동
- 이 단계에서 제목, 서브제목, 핵심 투자 포인트, OG 메타 등을 **편집**합니다 (→ [7장](#7-im-승인-및-편집-고도화) 참조)

## 4. 7섹션 구성과 각 섹션 설명

### 섹션 1: 🏢 이 건물, 어떤 자산인가?
| 포함 정보 | 데이터 출처 |
|-----------|-----------|
| 자산 유형, 권역, 규모 | 브로커 메모 + AI |
| 건축물대장 요약 | 국토부 API |
| 대지면적, 연면적, 용적률, 건폐율 | 건축물대장 API |

### 섹션 2: 📍 입지와 접근성은?
| 포함 정보 | 데이터 출처 |
|-----------|-----------|
| 최근접 지하철역/거리 | 카카오 POI API |
| 용도지역/용도지구 | 토지이음 API |
| 공시지가 | 공시지가 API |
| 위치 지도 | **카카오 로컬 API 지오코딩** + 카카오맵 |

### 섹션 3: 💰 현재 임대차 현황은?
| 포함 정보 | 데이터 출처 |
|-----------|-----------|
| 층별 임대차 현황 (Rent Roll) | 브로커 직접 입력 |
| WALE 자동 계산 | 자동 계산 |

### 섹션 4: 📊 수익 구조 분석
NOI, Cap Rate, 레버리지 수익률, 10년 DCF 분석, WACC, 자기자본 소요액

### 섹션 5: ⚠️ 숨은 리스크는 없는가?
리스크 요인 분석, 실사 체크리스트, 교차검증

### 섹션 6: 🎯 왜 지금 이 매물을 사야 하는가?
투자 가설, Value-Add 시나리오

### 섹션 7: 🔧 검토 후 다음 단계는?
다음 절차 안내, 중개인 연락처, 프라이빗 IM 신청 CTA

## 5. 히어로 카드 이해하기

히어로 카드는 IM 뷰어 최상단에 표시되는 **핵심 투자 지표 요약 카드**입니다.

| 지표 | 의미 | 산출 근거 |
|------|------|---------| 
| Cap Rate | 기본 수익률 | NOI ÷ 매각가 |
| NOI | 순영업이익 | (월세×12) - 관리비 |
| 자기자본 소요 | 실제 투자금 | 매각가 - 대출 - 보증금 |
| 레버리지 수익률 | 실질 수익률 | NOI ÷ 자기자본 |
| DCF NPV | 10년 순현재가치 | 할인현금흐름 분석 |

### 💡 핵심 투자 포인트 (v4.0 NEW)

히어로 카드 하단에 **💡 핵심 투자 포인트**가 그래디언트 박스로 강조 표시됩니다.

- AI가 자동 생성하지만, **IM 승인 화면에서 직접 편집** 가능 (v4.0)
- 투자 결정에 가장 중요한 1~2문장으로 작성 권장
- 예시: *"강남 핵심 상권 내 대로변 코너 입지로, 안정적인 임대 수익과 향후 개발 잠재력을 동시에 갖춘 투자 적격 자산입니다."*

## 7. IM 승인 및 편집 (고도화)

IM 생성 후 자동으로 **승인 페이지**(`/broker/im-approval/[id]`)로 이동합니다.
이 페이지는 발행 전 **투자 요약, OG 공유 정보, 섹션 내용**을 편집하는 핵심 화면입니다.

### 7-1. 📋 투자 요약 편집 (v4.0 NEW) ⭐

승인 페이지 상단에 새로운 **"📋 투자 요약 편집"** 섹션이 추가되었습니다.

이 섹션에서 **공개 IM 뷰어에 표시될 핵심 정보 3가지**를 직접 편집할 수 있습니다:

#### ① IM 제목 (공유 화면에 표시)

- **큰 텍스트 입력** (밑줄 스타일)으로 표시됩니다
- AI가 자동 생성한 제목을 자유롭게 수정 가능
- **업계 표준 문체 권장** — `"강남역 초역세권 프라임 꼬마빌딩 매각"` 스타일
- 플레이스홀더: `예: 강남역 초역세권 프라임 오피스 매각`

> ⚠️ **중요**: 이 제목은 공개 IM 뷰어의 `<h1>` 제목, 카카오톡 공유 OG 제목, 소통 관리함 알림에 모두 반영됩니다.

> 💡 **문체 참고**: 아래 [14장 IM 제목 문체 가이드 (골든셋)](#14-im-제목-문체-가이드-골든셋)에서 자산 유형별 20개 표준 예시를 확인하세요.

#### ② 서브제목 (핵심 투자 하이라이트 헤드카피)

- **에메랄드 색상** 텍스트로 가격대 아래에 표시됩니다
- 핵심 투자 하이라이트를 **한 줄 헤드카피**로 요약
- 플레이스홀더: `예: 연 수익률 6.4% · 공실 Zero · 우량 임차 기반 안정 수익형`
- 미입력 시 공개 IM에 서브제목이 표시되지 않습니다

> 💡 **좋은 서브제목 작성법**: 가격이 아닌 **투자 매력 포인트**를 중심으로 작성하세요.
> - ✅ `"연 수익률 5.2% · 공실 Zero · 장기 임차 기반 안정 수익형"`
> - ✅ `"리모델링 밸류애드 · 1층 F&B 프리미엄 · 시세 대비 저평가"`
> - ❌ `"매각 희망가 65억 원, 제안 가격 55억 원"` (가격만 나열)

#### ③ 💡 핵심 투자 포인트

- **그래디언트 박스** 내에 textarea로 표시됩니다
- AI가 자동 생성한 내용을 직접 수정 가능
- 1~2문장으로 **왜 이 매물에 투자해야 하는지** 명확히 서술
- 플레이스홀더: `예: 강남 핵심 상권 내 대로변 코너 입지로, 안정적인 임대 수익과 향후 개발 잠재력을 동시에 갖춘 투자 적격 자산입니다.`

#### 저장 방법
1. 제목, 서브제목, 핵심 투자 포인트를 편집합니다
2. 미저장 변경이 있으면 우측에 `⚠️ 저장되지 않음` 뱃지가 표시됩니다
3. **"💾 투자 요약 저장"** 버튼을 탭합니다 (보라색 그래디언트 버튼)
4. 저장 성공 시 `⚠️` 뱃지가 사라집니다

> 💡 **투자 요약 저장은 OG 저장과 별도입니다.** 두 섹션 모두 저장해야 완전 반영됩니다.

### 7-2. 🖼️ 공유 OG 메타 및 이미지 관리 (v4.0 개선) ⭐

투자 요약 편집 아래에 **"🖼️ 공유 OG 메타 및 이미지 관리"** 섹션이 있습니다.

#### 좌측: OG 이미지 미리보기 (실시간)

- 1200×630 정규 OG 이미지가 실시간으로 렌더링됩니다
- OG 타이틀/설명을 변경하고 저장하면 미리보기가 즉시 갱신됩니다
- 이 이미지가 **카카오톡, LINE, 문자 등에 공유 시** 표시됩니다

#### 우측: OG 메타 입력

| 필드 | 설명 | 카카오톡에서의 표시 |
|------|------|-----------------|
| **OG 타이틀 (카톡 공유 제목)** | 카카오톡 공유 시 표시되는 제목 | 링크 카드의 **제목** |
| **OG 설명 (카톡 공유 설명)** | 카카오톡 공유 시 표시되는 부가 설명 | 링크 카드의 **설명** 텍스트 |

#### v4.0 핵심 변경사항

> ⚠️ **이전 버전 문제**: OG 타이틀/설명을 편집해도 카카오톡 SDK 공유 시 **하드코딩된 기본값**이 사용되어 커스텀 메타가 반영되지 않았습니다.

> ✅ **v4.0 수정**: 카카오톡 SDK `sendDefault()` 호출 시 저장된 `ogTitle`, `ogDescription`을 **우선 사용**하도록 변경되었습니다. 편집한 OG 메타가 그대로 카카오톡에 표시됩니다.

#### 저장 방법
1. OG 타이틀과 OG 설명을 입력합니다
2. `⚠️ 저장되지 않음`이 표시됩니다
3. **"💾 OG 정보 저장"** 버튼을 탭합니다
4. 좌측 OG 이미지 미리보기가 자동 새로고침됩니다

> 💡 **OG 이미지**: 건물 사진이 아닌 **1200×630 정규 OG 이미지**가 자동 생성됩니다. 이 이미지에는 권역, 가격대, 제목, 설명, 데이터 품질 뱃지가 포함되어 카카오톡에서 잘리지 않고 정상 표시됩니다. (v4.0 수정)

## 10. OG 공유 메타 관리 및 카카오톡 공유 (v4.0 대폭 개선) ⭐

### OG 이미지란?

**OG(Open Graph) 이미지**는 카카오톡, LINE, 문자, 슬랙 등에 URL을 공유했을 때 자동으로 표시되는 **미리보기 이미지**입니다.

크리딜은 각 IM/딜카드마다 **1200×630px 정규 OG 이미지**를 자동 생성합니다.

### v4.0 OG 관련 주요 수정사항

#### 🔴 딜카드 OG 이미지 잘림 문제 해결

| 항목 | v3.0 (이전) | v4.0 (현재) |
|------|------------|------------|
| **OG 이미지 소스** | 건물 원본 사진 우선 → 비정형 크기 → 카카오톡에서 **잘림** | 항상 **1200×630 정규 OG 이미지** 사용 |
| **카카오톡 표시** | 사진이 잘리거나 검은 여백 발생 | 정상 비율로 깔끔하게 표시 |

#### 🔴 IM 카카오톡 공유 시 OG 메타 미반영 문제 해결

| 항목 | v3.0 (이전) | v4.0 (현재) |
|------|------------|------------|
| **카카오 SDK 제목** | 하드코딩: `[권역] 블라인드명` | 저장된 **ogTitle** 우선 사용 |
| **카카오 SDK 설명** | 하드코딩: `매각 희망가 X · Cap Rate Y%` | 저장된 **ogDescription** 우선 사용 |
| **편집한 메타 반영** | ❌ 미반영 | ✅ 즉시 반영 |

### OG 이미지에 포함되는 정보

자동 생성되는 1200×630 OG 이미지에는 다음 정보가 표시됩니다:

| 요소 | 설명 |
|------|------|
| **상단 뱃지** | 권역 레이블, 가격대, 데이터 품질 등급 |
| **중앙 제목** | 커스텀 ogTitle → IM 제목 → 자산 유형 (우선순위) |
| **서브타이틀** | 커스텀 ogDescription → IM 요약 → 기본 포맷 |
| **하단 지표** | 자산 유형, Cap Rate, NOI (있는 경우) |
| **브랜딩** | "Mobile IM" (IM) 또는 "DealCard" (딜카드) |
| **완성도 바** | 데이터 충실도 점수 시각화 |

### 카카오톡 공유 방법

#### ① SDK 공유 (권장)
1. IM 뷰어 하단의 **🟡 카카오톡** 버튼 탭
2. 카카오톡 앱이 열리며 대화 상대 선택
3. 리치 메시지가 전송됩니다:
   - **이미지**: 1200×630 OG 이미지
   - **제목**: 편집한 ogTitle (또는 IM 제목)
   - **설명**: 편집한 ogDescription (또는 자동 생성 요약)
   - **버튼**: "투자설명서 즉시 열람" → IM 뷰어로 이동

#### ② URL 직접 공유
1. **📋 링크복사** 버튼으로 URL 복사
2. 카카오톡에 직접 붙여넣기
3. 카카오 크롤러가 페이지의 `og:image`, `og:title`, `og:description` 태그를 파싱하여 표시

#### ③ LINE 공유
1. **🟢 LINE** 버튼 탭
2. LINE 앱이 열리며 공유

#### ④ PDF 저장
1. **📄 PDF 저장** 버튼 탭
2. IM 내용이 PDF로 변환되어 다운로드

### 카카오 OG 캐시 초기화

OG 타이틀/설명을 변경한 후 카카오톡에 **즉시 반영**하려면:

1. [카카오 OG 디버거](https://developers.kakao.com/tool/debugger/sharing) 접속
2. IM URL 입력 (예: `https://credeal.net/im-lite/{buildingId}`)
3. **"캐시 초기화"** 클릭
4. 이후 공유하면 새 정보가 반영됩니다

> ⚠️ 카카오 OG 디버거를 사용하지 않으면, 기존 캐시가 만료될 때까지 (수 시간~1일) 이전 정보가 보일 수 있습니다.

### 공유 URL 구조

| 콘텐츠 | 공유 URL | OG 이미지 소스 |
|--------|---------|-------------|
| **딜카드** | `https://credeal.net/dc/{buildingId}` | `/api/og/deal/{buildingId}` (1200×630) |
| **모바일 IM** | `https://credeal.net/im-lite/{buildingId}` | `/api/og/deal/{buildingId}?type=im` (1200×630) |

| 패턴 | 처리 | 예시 |
|------|------|------|
| `"~로 추정되는"` | 자동 제거 | `"근린생활시설로 추정되는"` → `"근린생활시설"` |
| `"또는 ~계열로 추정"` | 자동 제거 | `"또는 다가구·상가주택 계열로 추정"` → 삭제 |
| `"~ 투자설명서"` 접미사 | `"~ 매각"`으로 변경 | `"... 투자설명서"` → `"... 매각"` |

생성된 제목은 승인 화면에서 자유롭게 수정할 수 있습니다.

## 15. 자주 묻는 질문

### Q: IM 생성에 얼마나 걸리나요?
A: **30초~60초** 소요. Vercel Pro에서 최대 5분까지 처리됩니다.

### Q: 생성된 IM을 수정할 수 있나요?
A: 네. 승인 페이지에서 **제목, 서브제목, 핵심 투자 포인트, OG 메타, 섹션 내용** 모두 편집 가능합니다. PC는 Split-pane, 모바일은 편집/미리보기 탭 전환.

### Q: IM 제목이 어색하게 생성됩니다.
A: v4.0에서 `"~계열로 추정"`, `"또는 ~"` 등의 분석적 어투가 자동 제거되도록 개선되었습니다. 그래도 어색하다면 **투자 요약 편집** 섹션에서 직접 수정하세요. [14장의 골든셋](#14-im-제목-문체-가이드-골든셋)을 참고하면 전문적인 제목을 작성할 수 있습니다.

### Q: 서브제목은 무엇인가요?
A: v4.0에서 추가된 **핵심 투자 하이라이트 헤드카피**입니다. 가격대 아래에 에메랄드 색상으로 표시되며, 투자 매력 포인트를 한 줄로 요약합니다. 미입력 시 표시되지 않습니다.

### Q: 카카오톡으로 공유했는데 편집한 제목/설명이 안 나와요.
A: 두 가지를 확인하세요:
1. **IM 승인 화면에서 "💾 OG 정보 저장" 클릭 확인** — 저장하지 않으면 반영되지 않습니다
2. **카카오 OG 캐시 초기화** — [카카오 디버거](https://developers.kakao.com/tool/debugger/sharing)에서 해당 URL의 캐시를 초기화한 후 다시 공유하세요

### Q: OG 이미지가 잘려서 표시됩니다.
A: v4.0에서 해결되었습니다. 이전에는 건물 원본 사진(비정형 크기)이 OG 이미지로 사용되어 잘렸지만, 이제 **1200×630 정규 OG 이미지**가 항상 사용됩니다. 기존 딜카드도 자동 적용됩니다.

### Q: 음성 브리핑이 안 나와요.
A: v4.0에서 **TTS 안정화 전까지 음성 브리핑 기능이 일시 제거**되었습니다. 안정화 후 재도입 예정입니다.

### Q: 특정 섹션을 숨길 수 있나요?
A: 네. 승인 페이지에서 각 섹션의 **👁 토글**을 누르면 공개 IM에서 숨겨집니다.

### Q: 사진에 캡션을 달 수 있나요?
A: 네. 승인 편집 페이지에서 각 사진 아래 캡션 입력란에 텍스트를 입력하면 됩니다.

### Q: IM을 삭제할 수 있나요?
A: 네. IM 보관함에서 🗑 버튼으로 삭제. 영구적이며 복구 불가.

### Q: 지도에 정확한 위치가 표시되나요?
A: 네. **카카오 로컬 API**를 사용하여 한국 주소를 정확하게 지오코딩합니다.

### Q: 투자 요약 저장과 OG 정보 저장은 다른 건가요?
A: 네. 두 개의 **독립된 저장 영역**입니다.
- **📋 투자 요약 저장**: IM 제목, 서브제목, 핵심 투자 포인트 → 공개 IM 뷰어에 반영
- **🖼️ OG 정보 저장**: OG 타이틀, OG 설명 → 카카오톡/SNS 공유 시 표시되는 메타 정보에 반영
- 둘 다 저장해야 완전 반영됩니다
