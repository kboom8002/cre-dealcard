## 1. 꼬마빌딩 전문 중개인 (JS부동산) — PoC 참가 동기 재평가

### 솔직한 진단

> **참가 동기: 있다.** 단, "AI가 대단해서"가 아니라 **"반복 노동을 줄여줘서 + 카톡에서 명함이 눈에 띄어서"**

꼬마빌딩 전문 중개인의 하루:
```
09:00  건물주 전화 → 매물 메모 작성
09:30  정부24 접속 → 건축물대장 출력 → 5분 헤맴
09:45  세움터 접속 → 토지이용계획 확인 → 3분 헤맴
10:00  밸류맵/네이버 → 실거래가 확인
10:20  엑셀/한글에 블라인드 매물장 작성 → 주소 빼야 하는데 실수 위험
10:40  카톡 그룹방에 텍스트로 매물 공유 → 누구도 안 읽음
11:00  매수자 후보 한 명 한 명 전화
```

**시스템이 정말 아끼는 시간**: 09:30~10:40의 **70분을 5분**으로. 이것이 핵심 소구점.

### 🏆 탑 5 소구력

#### 🥇 1위: 건축물대장 + 실거래가 + 토지이용계획 원클릭 조회 (빌딩 레이더)

```
소구력:  ██████████  10/10
동기:    반복 고통 제거 — 매물마다 반드시 하는 작업
```

| 왜 강한가 | 왜 약할 수 있는가 |
|---------|----------------|
| 매일 하는 작업 → 체감 즉각적 | 정부 API 장애 시 빈 결과 → 신뢰 하락 |
| 정부24/세움터 UX가 끔찍 → 대안 갈망 | 밸류맵이 유사 기능 일부 제공 |
| 주소 1개 입력으로 5개 데이터 소스 통합 | — |

> **핵심 한마디**: "정부24 안 들어가도 됩니다."  
> **비판적 검증**: 이 기능 하나만으로도 PoC 참여 이유가 됨. **밸류맵과의 차이는 "딜카드에 자동 통합"**

## 2. 일반 집합건물 전문 중개인 — PoC 참가 동기 재평가

### 솔직한 진단

> **참가 동기: 제한적.** 전체 업무의 ~30%만 시스템과 접점이 있다.  
> 단, **"구분상가 + 상가 공실 임대"가 있는 중개인**에게는 의미 있음.

```
집합건물 중개인의 업무 구성:
  아파트/빌라 매매·임대  ████████████░░░░░░░░  60%  ← 시스템 접점 없음
  오피스텔 매매·임대      ████░░░░░░░░░░░░░░░░  20%  ← 부분 접점
  구분상가 매매           ███░░░░░░░░░░░░░░░░░  15%  ← 접점 있음
  상가 공실 임대          ██░░░░░░░░░░░░░░░░░░   5%  ← 접점 있음
                                                 ────
                              시스템 활용 비중:  ~30%
```

> [!WARNING]
> **핵심 문제**: 70%의 업무(아파트/빌라)와 무관한 도구를 "매일" 쓸 동기가 없다. **"가끔 유용"은 습관이 되지 않는다.**

### 🏆 탑 5 소구력

#### 🥇 1위: Vibe AI 모바일 명함 — 범용적 브랜딩 도구

```
소구력:  ████████░░  8/10
동기:    자기 브랜딩 — 업종 무관하게 작동
```

| 왜 강한가 | 왜 약할 수 있는가 |
|---------|----------------|
| **아파트/빌라 중개에서도 사용 가능** → 유일한 범용 기능 | 집합건물 중개사도 "명함? 카톡 프사로 충분한데" |
| 고객(세입자/매수자)에게 카톡으로 공유 → 첫인상 차별화 | 20~30대 젊은 중개인에게만 어필 가능 |
| 거래 실적이 표시되면 "경험 많은 중개사" 어필 | 초기에 실적 데이터가 없음 |

> **이것이 1위인 이유**: 집합건물 중개인의 업무 70%인 아파트/빌라에서도 쓸 수 있는 **유일한 기능**. 딜카드나 빌딩 레이더는 꼬마빌딩/상가에서만 작동하지만, 명함은 **모든 거래에서 사용 가능**.

## 3. CRE 전문직 (변호사, 회계사, 법무사 등) — Vendor 참여 동기

### 솔직한 진단

```
전문직의 현실:
  · 지인 소개로 먹고 산다 (90%)
  · 로톡/삼쩜삼 등 기존 플랫폼도 '해봤는데 별로' 경험
  · "또 하나의 플랫폼에 등록해야 하나?" 피로감
  · 다만 "무료"라면 일단 등록은 함 → 하지만 관리 안 함
```

### 🏆 탑 5 소구력 (DealCard Hub 전체에서)

#### 🥇 1위: Agora 전문가 배지 — CRE 니치 SEO 선점

```
소구력:  ████████░░  8/10
동기:    "상업용 부동산 전문 세무사/변호사" 포지셔닝
```

| 왜 강한가 | 왜 약할 수 있는가 |

## 4. 꼬마빌딩 및 대지 소유주 — 참여 동기

### 솔직한 진단

> **참여 동기: 가장 약하다.** 건물주/대지 소유주는 "도구를 쓰는 사람"이 아니라 "서비스를 받는 사람"이다.

```
건물주의 현실:
  · 부동산 거래는 1~2년에 1번 (고빈도 사용자가 아님)
  · 이미 담당 중개사가 있음 → 새 플랫폼 탐색 동기 없음
  · "앱을 깔아서 매물을 올린다"는 행위 자체가 부자연스러움
  · 시스템과의 자연스러운 접점:
    - 중개인이 보낸 딜카드/명함을 "열람"하는 수동적 참여
    - 빌딩 레이더로 자기 건물 가치를 "조회"하는 일회성 접점
    - 아고라에서 매각/세금 관련 질문을 "하는" 정보 소비
```

> [!WARNING]
> **건물주를 "PoC 참여자"로 모집하는 것은 비추.** 건물주는 **플랫폼의 최종 수혜자**이지, 능동적 사용자가 아님. 건물주는 중개인이 보낸 딜카드/명함을 통해 **자연스럽게 유입**되어야 함.

### 🏆 탑 5 소구력 (수동적 참여 기준)

#### 🥇 1위: 빌딩 레이더 — "내 건물이 지금 얼마일까?"

```
소구력:  █████████░  9/10
동기:    자산 가치 확인 — 건물주의 본능적 관심사
```

| 왜 강한가 | 왜 약할 수 있는가 |
|---------|----------------|
| **모든 건물주가 1순위로 궁금한 것** = "내 건물 시세" | 주소 입력만으로 가치 평가 → 정확도 한계 |
| 공시지가+실거래가+건축물대장 통합 리포트 | R-ONE/밸류맵으로도 비슷한 정보 확인 가능 |
| "무료로 한번 해보세요" → 바이럴 트리거 | 1회 조회 후 재방문 동기 부재 |
| 조회 → 매각 관심 시그널로 활용 가능 | — |

> **비판적 검증**: 빌딩 레이더는 건물주를 **"1회 방문자"에서 "관심 잠재 고객"으로 전환**하는 퍼널의 상단. 하지만 **1회 조회 이후 재방문 이유가 없음.** "매각 준비도 체크"나 "정기 가치 업데이트 알림"으로 후속 연결해야 함.

> [!IMPORTANT]
> ### 가장 중요한 발견
> 
> **Vibe AI 명함이 "범용 소구 도구"로 부상했다.** 꼬마빌딩 중개인에게는 2위, 집합건물 중개인에게는 1위이다. 이유는 명함만이 **아파트/빌라를 포함한 모든 거래에서 사용 가능한 유일한 기능**이기 때문이다.
> 
> 반면, 빌딩 레이더·딜카드·파이프라인 등 핵심 기능은 **꼬마빌딩/상가에만 작동**하여 집합건물 중개인의 주 업무(아파트)와 단절된다.
> 
> 따라서 **PoC 확장 시, Vibe AI 명함을 범용 진입점(entry funnel)으로 활용하고, 상업용 특화 기능은 해당 거래가 있는 사용자에게만 활성화하는 점진적 온보딩**이 필요하다.
